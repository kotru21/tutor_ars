# Метод интервалов

<LessonSection title="Метод интервалов: общий алгоритм">

**Метод интервалов** — это универсальный способ решения неравенств вида:

- $P(x) > 0$ (или $\geq 0$, $< 0$, $\leq 0$)
- Где $P(x)$ — произведение или частное линейных множителей

**Алгоритм решения:**

1. **Найти нули функции** — точки, где каждый множитель равен нулю
2. **Отметить эти точки на числовой прямой**, разбивая её на интервалы
3. **Определить знак функции на каждом интервале** (методом чередования с учетом кратности корней)
4. **Выбрать нужные интервалы** в соответствии со знаком неравенства
5. **Записать ответ** с учётом строгости неравенства

</LessonSection>

<LessonSection title="Пример 1: Решение неравенства">

**Решение неравенства:**

$$(x - 1)(x + 2,5)(x - 4) < 0$$

---

**Шаг 1: Находим нули функции**

$$x - 1 = 0 \quad\Rightarrow\quad x = 1$$

$$x + 2,5 = 0 \quad\Rightarrow\quad x = -2,5$$

$$x - 4 = 0 \quad\Rightarrow\quad x = 4$$

---

**Шаг 2: Отмечаем точки на числовой прямой**

Все корни имеют нечетную кратность (1), значит, знак будет чередоваться при переходе через каждый корень.

![Числовая прямая с точками -2.5, 1, 4](https://i.postimg.cc/ZRfhqGS4/изображение.png)

---

**Шаг 3: Определяем знаки методом чередования**

На интервале $(4, +\infty)$ подставим, например, $x = 5$:
$$(5-1)(5+2,5)(5-4) = 4 \cdot 7,5 \cdot 1 > 0$$
→ знак "+"

Двигаемся слева направо, чередуя знаки при переходе через каждый корень:

1. $(-\infty, -2,5)$: знак "-"
2. $(-2,5, 1)$: знак "+"
3. $(1, 4)$: знак "-"
4. $(4, +\infty)$: знак "+"

---

**Шаг 4: Выбираем интервалы, где выражение $< 0$**

Нас интересуют интервалы, где выражение отрицательно:

$$x \in (-\infty, -2,5) \cup (1, 4)$$

![Итоговое решение методом интервалов](https://i.postimg.cc/3xMQWjPd/изображение.png)

---

**Шаг 5: Ответ**

$$x \in (-\infty, -2,5) \cup (1, 4)$$

</LessonSection>

<LessonSection title="Пример 2: Решение неравенства">

**Дано неравенство:** $(x+9)^2(x-2)(x-3) \leq 0$

**Особенности:**

1. Множитель $(x+9)^2$ имеет чётную степень (квадрат)
2. Неравенство нестрогое ($\leq$), поэтому точки, где выражение равно нулю, включаются в ответ

**Решение:**

**Шаг 1: Находим нули функции**

$$(x+9)^2 = 0$$
$$x = -9$$ (чётная кратность)

$$x - 2 = 0$$
$$x = 2$$ (нечётная кратность)

$$x - 3 = 0$$
$$x = 3$$ (нечётная кратность)

**Шаг 2: Отмечаем точки на числовой прямой**

$$-9 \quad 2 \quad 3$$

**Шаг 3: Определяем знаки методом чередования с учётом кратности**

На самом правом интервале $(3, +\infty)$ все множители положительны → знак "+"

1. При переходе через $x=3$ (нечётная кратность) меняем знак на "-"
2. При переходе через $x=2$ (нечётная кратность) меняем знак на "+"
3. При переходе через $x=-9$ (чётная кратность) знак НЕ меняется

Получаем:

1. $(-\infty, -9)$: знак "+"
2. $(-9, 2)$: знак "+" (знак не изменился при переходе через -9)
3. $(2, 3)$: знак "-"
4. $(3, +\infty)$: знак "+"

**Шаг 4: Выбираем интервалы, где выражение $\leq 0$**

Нас интересуют интервалы, где выражение отрицательно или равно нулю

![Итоговое решение методом интервалов](https://i.postimg.cc/qRv9W1CC/изображение.png)

**Шаг 5: Ответ**

$$x \in [2, 3] \cup \{-9\}$$

</LessonSection>

<LessonSection title="Пример 3: Решение дробно-рационального неравенства">

**Решите неравенство:**

$$\frac{(x+3)(x-1)}{(x-4)(x+2)} \geq 0$$

**Решение:**

**Шаг 1: Находим критические точки**

**Нули числителя (дробь равна 0):**

$$(x+3)(x-1) = 0$$
$$x = -3$$ (нечётная кратность)
$$x = 1$$ (нечётная кратность)

**Нули знаменателя (дробь не определена):**

$$(x-4)(x+2) = 0$$
$$x = 4$$ (нечётная кратность)
$$x = -2$$ (нечётная кратность)

**Шаг 2: Отмечаем точки на числовой прямой**

Все корни имеют нечётную кратность (1)

![Числовая прямая с точками -3, -2, 1, 4](https://i.postimg.cc/cCV2gZkR/изображение.png)

**Шаг 3: Определяем знаки методом чередования**

На самом правом интервале $(4, +\infty)$ все множители положительны → знак "+"

Двигаемся справа налево, чередуя знаки при переходе через каждую точку:

1. $(4, +\infty)$: знак "+"
2. $(1, 4)$: знак "-"
3. $(-2, 1)$: знак "+"
4. $(-3, -2)$: знак "-"
5. $(-\infty, -3)$: знак "+"

**Шаг 4: Выбираем интервалы, где дробь $\geq 0$**

Нас интересуют интервалы, где дробь положительна или равна нулю

![Итоговое решение методом интервалов](https://i.postimg.cc/59GZkq9s/изображение.png)

**Шаг 5: Ответ**

$$x \in (-\infty, -3] \cup (-2, 1] \cup (4, +\infty)$$

</LessonSection>

<LessonSection title="Пример 4: Решение дробно-рационального неравенства с чётной степенью">

**Решите неравенство:**

$$\frac{x(x-2)}{(x+1)^4} > 0$$

**Решение:**

**Шаг 1: Находим критические точки**

**Нули числителя:**

$$x(x-2) = 0$$
$$x = 0$$ (нечётная кратность)
$$x = 2$$ (нечётная кратность)

**Нули знаменателя:**

$$(x+1)^4 = 0$$
$$x = -1$$ (чётная кратность 4)

**Шаг 2: Отмечаем точки на числовой прямой**

$$-1 \quad 0 \quad 2$$

![Числовая прямая с точками -1, 0, 2](https://i.postimg.cc/T39xSL43/изображение.png)

**Шаг 3: Определяем знаки методом чередования с учётом кратности**

На самом правом интервале $(2, +\infty)$ все множители положительны → знак "+"

1. При переходе через $x=2$ (нечётная кратность) меняем знак на "-"
2. При переходе через $x=0$ (нечётная кратность) меняем знак на "+"
3. При переходе через $x=-1$ (чётная кратность) знак НЕ меняется

Получаем:

1. $(2, +\infty)$: знак "+"
2. $(0, 2)$: знак "-"
3. $(-1, 0)$: знак "+"
4. $(-\infty, -1)$: знак "+"

**Шаг 4: Выбираем интервалы, где дробь $> 0$**

![Итоговое решение методом интервалов](https://i.postimg.cc/nLSJV8xB/изображение.png)

**Шаг 5: Ответ**

$$x \in (-\infty, -1) \cup (-1, 0) \cup (2, +\infty)$$

</LessonSection>

<LessonSection title="Пример 5: Решение неравенства с преобразованием">

**Решите неравенство:**

$$\frac{(x-4)^2(3-x)}{x+1} \geq 0$$

**Решение:**

**Шаг 1: Преобразуем множитель $(3-x)$**

$$3-x = -(x-3)$$

Подставляем в неравенство:

$$\frac{(x-4)^2 \cdot [-(x-3)]}{x+1} \geq 0$$

**Выносим минус:**

$$-\frac{(x-4)^2(x-3)}{x+1} \geq 0$$

**Шаг 2: Умножаем обе части на -1 (меняем знак неравенства)**

$$\frac{(x-4)^2(x-3)}{x+1} \leq 0$$

**Шаг 3: Находим критические точки**

**Нули числителя:**

$$(x-4)^2 = 0$$
$$x = 4$$ (чётная кратность 2)

$$x-3 = 0$$
$$x = 3$$ (нечётная кратность)

**Нули знаменателя:**

$$x+1 = 0$$
$$x = -1$$ (нечётная кратность)

**Шаг 4: Отмечаем точки на числовой прямой**

$$-1 \quad 3 \quad 4$$

![Числовая прямая с точками -1, 3, 4](https://i.postimg.cc/MTc8vsKT/изображение.png)

**Шаг 5: Определяем знаки методом чередования с учётом кратности**

На самом правом интервале $(4, +\infty)$ все множители положительны → знак "+"

1. При переходе через $x=4$ (чётная кратность) знак НЕ меняется → остаётся "+"
2. При переходе через $x=3$ (нечётная кратность) меняем знак на "-"
3. При переходе через $x=-1$ (нечётная кратность) меняем знак на "+"

Получаем:

1. $(4, +\infty)$: знак "+"
2. $(3, 4)$: знак "+"
3. $(-1, 3)$: знак "-"
4. $(-\infty, -1)$: знак "+"

**Шаг 6: Выбираем интервалы, где выражение $\leq 0$**

![Итоговое решение методом интервалов](https://i.postimg.cc/kGK3yBb1/изображение.png)

**Шаг 7: Ответ**

$$x \in (-1, 3] \cup \{4\}$$

</LessonSection>

<LessonSection title="Пример 6: Решение сложного дробно-рационального неравенства">

**Решите неравенство:**

$$\frac{x}{x^2 - 5x + 6} + \frac{3}{x-2} > \frac{2}{x-3}$$

**Решение:**

**Шаг 1: Замечаем, что**

$$x^2 - 5x + 6 = (x-2)(x-3)$$

Преобразуем неравенство:

$$\frac{x}{(x-2)(x-3)} + \frac{3}{x-2} - \frac{2}{x-3} > 0$$

**Шаг 2: Приводим к общему знаменателю $(x-2)(x-3)$**

$$\frac{x + 3(x-3) - 2(x-2)}{(x-2)(x-3)} > 0$$

$$\frac{x + 3x - 9 - 2x + 4}{(x-2)(x-3)} > 0$$

$$\frac{2x - 5}{(x-2)(x-3)} > 0$$

**Шаг 3: Находим критические точки**

**Нули числителя:**

$$2x - 5 = 0$$
$$x = 2,5$$ (нечётная кратность)

**Нули знаменателя:**

$$(x-2)(x-3) = 0$$
$$x = 2$$ (нечётная кратность)
$$x = 3$$ (нечётная кратность)

**Шаг 4: Отмечаем точки на числовой прямой**

$$2 \quad 2,5 \quad 3$$

**Шаг 5: Определяем знаки методом чередования**

На самом правом интервале $(3, +\infty)$ все множители положительны → знак "+"

Двигаемся справа налево, чередуя знаки при переходе через каждую точку:

1. $(3, +\infty)$: знак "+"
2. $(2,5, 3)$: знак "-"
3. $(2, 2,5)$: знак "+"

**Шаг 6: Выбираем интервалы, где дробь $> 0$**

![Итоговое решение методом интервалов](https://i.postimg.cc/fbrGnx7B/изображение.png)

**Шаг 7: Ответ**

$$x \in (2, 2,5) \cup (3, +\infty)$$

</LessonSection>
