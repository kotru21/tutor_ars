# Дробно-рациональные неравенства

## Метод интервалов: общий алгоритм

<Definition>
**Метод интервалов** — это универсальный способ решения неравенств вида:
- $P(x) > 0$ (или $\geq 0$, $< 0$, $\leq 0$)
- Где $P(x)$ — произведение или частное линейных множителей
</Definition>

**Алгоритм решения:**

1. **Найти нули функции** — точки, где каждый множитель равен нулю
2. **Отметить эти точки на числовой прямой**, разбивая её на интервалы
3. **Определить знак функции на каждом интервале** (методом подстановки или анализа множителей)
4. **Выбрать нужные интервалы** в соответствии со знаком неравенства
5. **Записать ответ** с учётом строгости неравенства

---

## Пример 1: Произведение множителей

<Example>
**Решить неравенство:** $(x-1)(x+2{,}5)(x-4) < 0$

**Решение:**

1. Находим нули функции: каждый множитель приравниваем к нулю

$$x - 1 = 0 \Rightarrow x = 1$$

$$x + 2{,}5 = 0 \Rightarrow x = -2{,}5$$

$$x - 4 = 0 \Rightarrow x = 4$$

2. Отмечаем точки на числовой прямой в порядке возрастания: $-2{,}5$, $1$, $4$

3. Определяем знаки на интервалах:

- **$(-\infty, -2{,}5)$**: берём $x = -3$ → $(-)(-)(-)=(-) < 0$
- **$(-2{,}5, 1)$**: берём $x = 0$ → $(-)(+)(-)=(+) > 0$
- **$(1, 4)$**: берём $x = 2$ → $(+)(+)(-)=(-) < 0$
- **$(4, +\infty)$**: берём $x = 5$ → $(+)(+)(+)=(+) > 0$

4. Выбираем интервалы, где выражение < 0 (отрицательно).

**Ответ:** $x \in (-\infty, -2{,}5) \cup (1, 4)$

</Example>

---

## Пример 2: Чётные степени множителей

<Example variant="hard">
**Решить неравенство:** $(x+9)^2(x-2)(x-3) \leq 0$

**Решение:**

**Особенности:**

- Множитель $(x+9)^2$ имеет чётную степень (квадрат)
- Неравенство нестрогое ($\leq$), поэтому точки, где выражение равно нулю, включаются в ответ

1. Находим нули функции:

   $$
   (x+9)^2 = 0 \Rightarrow x = -9
   $$

   $$
   x - 2 = 0 \Rightarrow x = 2
   $$

   $$
   x - 3 = 0 \Rightarrow x = 3
   $$

2. **Правило смены знака:**

- В точке $x = -9$ (чётная кратность) — знак **НЕ меняется**
- В точках $x = 2$ и $x = 3$ (нечётная кратность) — знак **меняется**

3. Определяем знаки на интервалах:

- **$(-\infty, -9)$**: $(+)(-)(-) = (+) > 0$
- **$(-9, 2)$**: $(+)(-)(-) = (+) > 0$ (знак НЕ изменился!)
- **$(2, 3)$**: $(+)(+)(-) = (-) < 0$
- **$(3, +\infty)$**: $(+)(+)(+) = (+) > 0$

4. Выбираем интервалы, где выражение $\leq 0$:

- На интервале $(2, 3)$: выражение отрицательно
- В точках $x = -9$, $x = 2$, $x = 3$: выражение равно нулю

**Ответ:** $x \in [2, 3] \cup \{-9\}$

</Example>

<Explanation>
  **Объяснение:** Точка $-9$ включается, так как в этой точке выражение равно
  нулю. Хотя на интервалах $(-\infty, -9)$ и $(-9, 2)$ выражение положительно,
  сама точка $-9$ входит в решение.
</Explanation>

## Пример 3: Дробно-рациональное неравенство

<Example>
**Решить неравенство:** $\dfrac{(x+3)(x-1)}{(x-4)(x+2)} \geq 0$

**Решение:**

**Шаг 1: Находим критические точки**

Нули числителя (дробь равна 0):
$$(x+3)(x-1) = 0$$
$$x = -3 \text{ или } x = 1$$

Нули знаменателя (дробь не определена):
$$(x-4)(x+2) = 0$$
$$x = 4 \text{ или } x = -2$$

**Важно:** Нули числителя включаются в решение при нестрогом неравенстве ($\geq$), а нули знаменателя **никогда** не включаются (там функция не определена).

**Шаг 2: Отмечаем точки на числовой прямой**

Располагаем точки в порядке возрастания: $-3$, $-2$, $1$, $4$

- ● — закрашенная точка (включается в решение)
- ○ — выколотая точка (не включается в решение)

**Шаг 3: Определяем знаки на интервалах**

| Интервал        | Пробная точка | Знак дроби              |
| --------------- | ------------- | ----------------------- |
| $(-\infty, -3)$ | $x = -4$      | $(-)(-) / (-)(-) = (+)$ |
| $(-3, -2)$      | $x = -2{,}5$  | $(+)(-) / (-)(-)= (-)$  |
| $(-2, 1)$       | $x = 0$       | $(+)(-) / (-)(+) = (+)$ |
| $(1, 4)$        | $x = 2$       | $(+)(+) / (-)(+) = (-)$ |
| $(4, +\infty)$  | $x = 5$       | $(+)(+) / (+)(+) = (+)$ |

**Шаг 4: Записываем решение**

Нас интересуют интервалы, где дробь $\geq 0$ (положительна или равна нулю).

**Ответ:** $x \in (-\infty, -3] \cup (-2, 1] \cup (4, +\infty)$

</Example>

<Explanation>
  **Пояснение:** 1. **$(-\infty, -3]$** — включаем $-3$, так как при $x = -3$
  дробь равна $0$ 2. **$(-2, 1]$** — не включаем $-2$ (знаменатель = 0),
  включаем $1$ (числитель = 0) 3. **$(4, +\infty)$** — не включаем $4$
  (знаменатель = 0)
</Explanation>

## Пример 4: Чётная степень в знаменателе

<Example variant="hard">
**Решить неравенство:** $\dfrac{x(x-2)}{(x+1)^4} > 0$

**Решение:**

**Особенности:** В знаменателе $(x+1)^4$ — множитель в чётной степени.

1. Находим нули числителя и знаменателя:

Числитель: $x(x-2) = 0 \Rightarrow x = 0$ или $x = 2$

Знаменатель: $(x+1)^4 = 0 \Rightarrow x = -1$

2. Точка $x = -1$ (из знаменателя) всегда выколотая (не входит в область определения).

3. **Важно:** $(x+1)^4 > 0$ при всех $x \neq -1$, поэтому знак дроби определяется только числителем!

4. Определяем знаки на интервалах:

- **$(-\infty, -1)$**: $(-)(-) / (+) = (+) > 0$
- **$(-1, 0)$**: $(-)(-) / (+) = (+) > 0$
- **$(0, 2)$**: $(+)(-) / (+) = (-) < 0$
- **$(2, +\infty)$**: $(+)(+) / (+) = (+) > 0$

**Ответ:** $x \in (-\infty, -1) \cup (-1, 0) \cup (2, +\infty)$

</Example>

<Explanation>
  Точка $-1$ выколота (знаменатель равен $0$), точки $0$ и $2$ не включаются
  (строгое неравенство).
</Explanation>

## Пример 5: Преобразование множителя $(a - x)$

<Example variant="hard">
**Решить неравенство:** $\dfrac{(x-4)^2(3-x)}{x+1} \geq 0$

**Решение:**

**Шаг 1: Преобразуем множитель $(3-x)$, вынося минус:**
$$3 - x = -(x - 3)$$

**Шаг 2: Выносим минус перед всей дробью:**
$$-\frac{(x-4)^2(x-3)}{x+1} \geq 0$$

**Шаг 3: Умножаем обе части на $-1$ (меняем знак неравенства!):**
$$\frac{(x-4)^2(x-3)}{x+1} \leq 0$$

**Шаг 4: Находим критические точки:**

- Числитель = 0: $x = 4$ (чётная кратность), $x = 3$ (нечётная кратность)
- Знаменатель = 0: $x = -1$ (не входит в ОДЗ)

**Шаг 5: Определяем знаки:**

- $(-\infty, -1)$: $(+)(-)/(-)= (+) > 0$
- $(-1, 3)$: $(+)(-)/( +) = (-) < 0$
- $(3, 4)$: $(+)(+)/(+) = (+) > 0$
- $(4, +\infty)$: $(+)(+)/(+) = (+) > 0$

**Ответ:** $x \in (-1, 3] \cup \{4\}$

</Example>

<Explanation>
**Пояснение:**
1. **$(-1, 3]$** — точка $-1$ не входит (знаменатель = 0), точка $3$ входит (числитель = 0)
2. **$\{4\}$** — отдельная точка, где числитель = 0 (квадрат)
</Explanation>

## Пример 6: Приведение к общему знаменателю

<Example variant="hard">
**Решить неравенство:** $\dfrac{x}{x^2-5x+6} + \dfrac{3}{x-2} > \dfrac{2}{x-3}$

**Решение:**

1. Приведём к общему знаменателю. Заметим, что $x^2 - 5x + 6 = (x-2)(x-3)$.

2. Преобразуем неравенство:
   $$\frac{x}{(x-2)(x-3)} + \frac{3(x-3)}{(x-2)(x-3)} - \frac{2(x-2)}{(x-2)(x-3)} > 0$$

$$\frac{x + 3x - 9 - 2x + 4}{(x-2)(x-3)} > 0$$

$$\frac{2x - 5}{(x-2)(x-3)} > 0$$

3. Находим нули:

- Числитель: $2x - 5 = 0 \Rightarrow x = 2{,}5$
- Знаменатель: $x = 2$ или $x = 3$

4. Определяем знаки и записываем ответ.

**Ответ:** $x \in (2, 2{,}5) \cup (3, +\infty)$

</Example>

## Системы и совокупности неравенств

<LessonSection title="Система неравенств">

<Definition>
  **Система неравенств** — решение является **пересечением** решений отдельных
  неравенств (одновременное выполнение всех условий).
</Definition>

<Example>
**Решить систему:**
$$
\begin{cases}
(x-2)(x-3) \leq 0 \\
\dfrac{x-5}{(x-1)(x+1)} > 0
\end{cases}
$$

**Решение:**

1. Первое неравенство: $x \in [2, 3]$
2. Второе неравенство: $x \in (-1, 1) \cup (5, +\infty)$
3. Пересечение: $[2, 3] \cap [(-1, 1) \cup (5, +\infty)] = \varnothing$

**Ответ:** Система не имеет решений.

</Example>

</LessonSection>

<LessonSection title="Совокупность неравенств">

<Definition>
  **Совокупность неравенств** — решение является **объединением** решений
  отдельных неравенств (выполнение хотя бы одного условия).
</Definition>

<Example>
**Решить совокупность:**
$$
\left[\begin{gathered}
(x-3)(x-4)(x+5)^2 \geq 0 \\
2x - 12 < 0
\end{gathered}\right.
$$

**Решение:**

1. Первое неравенство: $x \in (-\infty, 3] \cup [4, +\infty)$
2. Второе неравенство: $x < 6$, т.е. $x \in (-\infty, 6)$
3. Объединение: $(-\infty, 3] \cup [4, +\infty) \cup (-\infty, 6) = (-\infty, +\infty)$

**Ответ:** $x \in \mathbb{R}$ (все действительные числа)

</Example>

</LessonSection>

## Итоги метода интервалов

<Definition>
  **Ключевые моменты:** 1. **Чётные степени** множителей не меняют знак при
  переходе через соответствующую точку 2. **Точки из знаменателя** всегда
  выколотые (не включаются в ответ) 3. **Точки из числителя** включаются при
  нестрогих неравенствах ($\geq$, $\leq$) 4. **Система неравенств** — находим
  пересечение решений 5. **Совокупность неравенств** — находим объединение
  решений
</Definition>

**Проверка решения:**

- Подставить пограничные значения (точки, где выражение = 0)
- Проверить знак на каждом интервале
- Убедиться, что учтены все ограничения (знаменатель $\neq 0$)
