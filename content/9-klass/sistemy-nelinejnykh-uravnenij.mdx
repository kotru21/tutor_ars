# Системы нелинейных уравнений

## Что такое система уравнений

<Definition>
**Система уравнений** — это набор двух или более уравнений, которые рассматриваются одновременно.

**Решение системы уравнений** — это упорядоченная пара чисел $(x; y)$, которая удовлетворяет всем уравнениям системы одновременно.

Геометрически решение системы представляет собой **точку пересечения графиков** уравнений, входящих в систему.

</Definition>

---

## Метод подстановки

<Definition>
  **Метод подстановки** — метод решения систем уравнений, при котором из одного
  уравнения выражают одну переменную через другую и подставляют это выражение в
  другое уравнение.
</Definition>

<Example>
**Решить систему уравнений:**

$$
\begin{cases}
x + y = 3 \\
x^2 + y^2 = 9
\end{cases}
$$

**Решение:**

1. Выразим $x$ из первого уравнения:
   $$x = 3 - y$$

2. Подставим это выражение во второе уравнение:
   $$(3 - y)^2 + y^2 = 9$$

3. Раскроем скобки и упростим:
   $$9 - 6y + y^2 + y^2 = 9$$
   $$2y^2 - 6y = 0$$
   $$2y(y - 3) = 0$$

4. Решим полученное уравнение:
   $$y = 0 \quad \text{или} \quad y = 3$$

5. Найдем соответствующие значения $x$:

При $y = 0$: $x = 3 - 0 = 3$

При $y = 3$: $x = 3 - 3 = 0$

**Ответ:** $(3; 0)$ и $(0; 3)$

</Example>

## Метод сложения

<Definition>
  **Метод сложения (исключения)** — метод решения систем уравнений, при котором
  уравнения системы складывают или вычитают таким образом, чтобы исключить одну
  из переменных.
</Definition>

<Example variant="hard">
**Решить систему уравнений:**

$$
\begin{cases}
y^2 + xy = -10 \\
5y - 2xy = 17
\end{cases}
$$

**Решение:**

1. Перепишем систему:

   $$
   \begin{cases}
   xy + y^2 = -10 \\
   -2xy + 5y = 17
   \end{cases}
   $$

2. Умножим первое уравнение на 2:
   $$2xy + 2y^2 = -20$$

3. Сложим полученное уравнение со вторым уравнением системы:
   $$(2xy + 2y^2) + (-2xy + 5y) = -20 + 17$$
   $$2y^2 + 5y = -3$$

4. Решим полученное квадратное уравнение:
   $$2y^2 + 5y + 3 = 0$$
   $$D = 25 - 24 = 1$$
   $$y_1 = \frac{-5 + 1}{4} = -1$$
   $$y_2 = \frac{-5 - 1}{4} = -\frac{3}{2}$$

5. Найдем соответствующие значения $x$ из первого уравнения системы:

При $y = -1$:
$$x \cdot (-1) + (-1)^2 = -10$$
$$-x + 1 = -10$$
$$x = 11$$

При $y = -\frac{3}{2}$:
$$x \cdot \left(-\frac{3}{2}\right) + \left(-\frac{3}{2}\right)^2 = -10$$
$$-\frac{3x}{2} + \frac{9}{4} = -10$$
$$-\frac{3x}{2} = -\frac{49}{4}$$
$$x = \frac{49}{6}$$

**Ответ:** $(11; -1)$ и $\left(\frac{49}{6}; -\frac{3}{2}\right)$

</Example>

## Графический метод

<Definition>
  **Графический метод** — метод решения систем уравнений, при котором строят
  графики уравнений, входящих в систему, и находят точки их пересечения.
</Definition>

<Example>
**Решить графически систему уравнений:**

$$
\begin{cases}
y - x - 2 = 0 \\
y - (x + 1)^2 = -1
\end{cases}
$$

**Решение:**

1. Преобразуем уравнения к виду, удобному для построения графиков:
   $$y = x + 2$$
   $$y = (x + 1)^2 - 1$$

2. Построим графики:

- Прямая $y = x + 2$ — линейная функция, проходящая через точки $(0; 2)$ и $(-2; 0)$
- Парабола $y = (x + 1)^2 - 1$ — парабола, ветви вверх, вершина в точке $(-1; -1)$

3. На графике видно две точки пересечения:
   $$A(-2; 0) \quad \text{и} \quad B(1; 3)$$

4. Проверим решение аналитически:

Для точки $A(-2; 0)$:
$$0 - (-2) - 2 = 0 + 2 - 2 = 0 \quad \checkmark$$
$$0 - (-2 + 1)^2 = 0 - (-1)^2 = 0 - 1 = -1 \quad \checkmark$$

**Ответ:** $(-2; 0)$ и $(1; 3)$

</Example>

## Советы по выбору метода

<LessonSection title="Когда использовать каждый метод">

**Метод подстановки** удобен, когда:

- Одна из переменных легко выражается из уравнения
- Одно из уравнений линейное

**Метод сложения** удобен, когда:

- Переменные входят с противоположными или пропорциональными коэффициентами
- Оба уравнения нелинейные, но имеют похожую структуру

**Графический метод** удобен, когда:

- Нужно оценить количество решений
- Уравнения легко строятся (прямые, параболы, окружности)
- Требуется визуальное представление

</LessonSection>

<Explanation>
  При решении систем нелинейных уравнений часто приходится комбинировать методы.
  Например, после сложения можно получить уравнение, которое решается
  подстановкой.
</Explanation>
