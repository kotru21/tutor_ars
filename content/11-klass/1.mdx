# **Типы логарифмических уравнений и методы их решения**

<LessonSection title="1. Простейшее логарифмическое уравнение">

**Уравнение вида:**

$$\log_a f(x) = b$$

---

**Алгоритм решения:**

**1.** Находим ОДЗ: $f(x) > 0$  
**2.** Переходим к показательной форме: $f(x) = a^b$  
**3.** Решаем полученное уравнение  
**4.** Проверяем соответствие ОДЗ

---

**Пример:**

$$\log_2(x - 3) = 0$$

---

**Решение:**

**ОДЗ:** $x - 3 > 0 \quad\Rightarrow\quad x > 3$

$$x - 3 = 2^0 = 1$$

$$x = 4$$

Проверка: $4 > 3$ — условие выполняется.

---

**Ответ:** $x = 4$

---

**Пример 2:**

$$\log_{27}(7 - x) - \frac{2}{3} = 0$$

---

**Решение:**

**ОДЗ:** $7 - x > 0 \quad\Rightarrow\quad x < 7$

$$\log_{27}(7 - x) = \frac{2}{3}$$

$$7 - x = 27^{2/3} = (27^{1/3})^2 = 3^2 = 9$$

$$7 - x = 9 \quad\Rightarrow\quad x = -2$$

Проверка: $-2 < 7$ — условие выполняется.

---

**Ответ:** $x = -2$

</LessonSection>

<LessonSection title="2. Уравнение с равенством логарифмов одного основания">

**Уравнение вида:**

$$\log_a f(x) = \log_a g(x)$$

---

**Алгоритм решения:**

**1.** Находим ОДЗ:  
 $f(x) > 0$  
 $g(x) > 0$  
**2.** Уравниваем аргументы: $f(x) = g(x)$  
**3.** Решаем полученное уравнение  
**4.** Проверяем ОДЗ для каждого корня

---

**Пример:**

$$\log_4(1 - x) = \log_4(x^2 + 8x - 9)$$

---

**Решение:**

**ОДЗ:**  
$1 - x > 0 \quad\Rightarrow\quad x < 1$  
$x^2 + 8x - 9 > 0$

$$1 - x = x^2 + 8x - 9$$

$$0 = x^2 + 9x - 10$$

$$x^2 + 9x - 10 = 0$$

$$D = 81 + 40 = 121$$

$$x_{1,2} = \frac{-9 \pm 11}{2}$$

$$x_1 = 1, \quad x_2 = -10$$

**Проверка ОДЗ:**  
$x = 1$: $1 - 1 = 0$ — не > 0 ❌  
$x = -10$: $1 - (-10) = 11 > 0$ ✅  
$(-10)^2 + 8 \cdot (-10) - 9 = 100 - 80 - 9 = 11 > 0$ ✅

---

**Ответ:** $x = -10$

---

**Пример 2:**

$$\log_6(x^2 + 6) = \log_6(5x)$$

---

**Решение:**

**ОДЗ:**  
$x^2 + 6 > 0$ — всегда верно  
$5x > 0 \quad\Rightarrow\quad x > 0$

$$x^2 + 6 = 5x$$

$$x^2 - 5x + 6 = 0$$

$$x_1 = 2, \quad x_2 = 3$$

Оба корня > 0 ✅

---

**Ответ:** $x = 2, 3$

</LessonSection>

<LessonSection title="3. Уравнение с логарифмическими преобразованиями">

**Уравнение содержит суммы/разности логарифмов:**

$$\log_a f(x) \pm \log_a g(x) = b$$

---

**Алгоритм решения:**

**1.** Применяем свойства логарифмов:  
 $\log_a f(x) + \log_a g(x) = \log_a [f(x) \cdot g(x)]$  
 $\log_a f(x) - \log_a g(x) = \log_a \frac{f(x)}{g(x)}$  
**2.** Учитываем ОДЗ каждого логарифма  
**3.** Сводим к типам 1 или 2  
**4.** Проверяем ОДЗ

---

**Пример:**

$$1 + \log_7(x + 4) = \log_7(x^2 + 9x + 20)$$

---

**Решение:**

**ОДЗ:**  
$x + 4 > 0 \quad\Rightarrow\quad x > -4$  
$x^2 + 9x + 20 > 0$

$$\log_7 7 + \log_7(x + 4) = \log_7(x^2 + 9x + 20)$$

$$\log_7[7(x + 4)] = \log_7(x^2 + 9x + 20)$$

Замечаем: $x^2 + 9x + 20 = (x + 4)(x + 5)$

$$7(x + 4) = (x + 4)(x + 5)$$

$$(x + 4)(7 - x - 5) = 0$$

$$(x + 4)(2 - x) = 0$$

$$x_1 = -4, \quad x_2 = 2$$

**Проверка ОДЗ:**  
$x = -4$: $x + 4 = 0$ — логарифм не определен ❌  
$x = 2$: $2 > -4$ ✅  
$x^2 + 9x + 20 = 4 + 18 + 20 = 42 > 0$ ✅

---

**Ответ:** $x = 2$

</LessonSection>

<LessonSection title="4. Уравнение с заменой переменной">

**Уравнение, где можно ввести $t = \log_a x$ или $t = \log_a f(x)$**

---

**Алгоритм решения:**

**1.** Вводим замену $t = \log_a x$  
**2.** Решаем алгебраическое уравнение относительно $t$  
**3.** Возвращаемся к исходной переменной: $x = a^t$  
**4.** Проверяем ОДЗ исходного уравнения

---

**Пример:**

$$\log_2 x^3 + 8\log_2 \sqrt{x} = -21$$

---

**Решение:**

**ОДЗ:** $x > 0$

Упрощаем:  
$\log_2 x^3 = 3\log_2 x$  
$\log_2 \sqrt{x} = \log_2 x^{1/2} = \frac{1}{2}\log_2 x$  
$8\log_2 \sqrt{x} = 8 \cdot \frac{1}{2}\log_2 x = 4\log_2 x$

$$3\log_2 x + 4\log_2 x = 7\log_2 x = -21$$

$$\log_2 x = -3$$

$$x = 2^{-3} = \frac{1}{8}$$

Проверка: $\frac{1}{8} > 0$ ✅

---

**Ответ:** $x = \frac{1}{8}$

</LessonSection>

<LessonSection title="5. Уравнение с разными основаниями">

**Уравнение вида:**

$$\log_{a} f(x) = \log_{b} g(x)$$

---

**Алгоритм решения:**

**1.** Находим ОДЗ:  
 $f(x) > 0$  
 $g(x) > 0$  
**2.** Приводим к одному основанию через свойство степени в основании:  
 $\log_{a^k} b = \frac{1}{k} \log_a b$  
**3.** Сводим к типам 1 или 2  
**4.** Проверяем ОДЗ

---

**Пример:**

$$\log_{25}(16 - 5x) = \log_5(2x - 5)$$

---

**Решение:**

**ОДЗ:**  
$16 - 5x > 0 \quad\Rightarrow\quad x < 3.2$  
$2x - 5 > 0 \quad\Rightarrow\quad x > 2.5$  
Итого: $2.5 < x < 3.2$

Приводим к основанию 5:  
$\log_{25}(16 - 5x) = \log_{5^2}(16 - 5x) = \frac{1}{2} \log_5(16 - 5x)$

$$\frac{1}{2} \log_5(16 - 5x) = \log_5(2x - 5)$$

$$\log_5(16 - 5x) = 2\log_5(2x - 5)$$

$$\log_5(16 - 5x) = \log_5(2x - 5)^2$$

$$16 - 5x = (2x - 5)^2$$

$$16 - 5x = 4x^2 - 20x + 25$$

$$4x^2 - 15x + 9 = 0$$

$$D = 225 - 144 = 81$$

$$x_{1,2} = \frac{15 \pm 9}{8}$$

$$x_1 = 3, \quad x_2 = 0.75$$

**Проверка ОДЗ ($2.5 < x < 3.2$):**  
$x = 3$: $3 \in (2.5, 3.2)$ ✅  
$x = 0.75$: $0.75 < 2.5$ ❌

---

**Ответ:** $x = 3$

</LessonSection>

<LessonSection title="6. Рациональное уравнение с логарифмами">

**Уравнение вида:**

$$\frac{P(\log_a x)}{Q(\log_a x)} = R(\log_a x)$$

---

**Алгоритм решения:**

**1.** Вводим замену $t = \log_a x$  
**2.** Решаем рациональное уравнение относительно $t$  
**3.** Возвращаемся к $x$: $x = a^t$  
**4.** Проверяем ОДЗ (знаменатели ≠ 0)

---

**Пример:**

$$\frac{1}{5 - 4\log_2 x} + \frac{4}{1 + \log_2 x} = 3$$

---

**Решение:**

**ОДЗ:**  
$x > 0$  
$5 - 4\log_2 x \neq 0 \quad\Rightarrow\quad \log_2 x \neq \frac{5}{4} \quad\Rightarrow\quad x \neq 2^{5/4}$  
$1 + \log_2 x \neq 0 \quad\Rightarrow\quad \log_2 x \neq -1 \quad\Rightarrow\quad x \neq \frac{1}{2}$

Замена: $t = \log_2 x$

$$\frac{1}{5 - 4t} + \frac{4}{1 + t} = 3$$

Умножаем на $(5 - 4t)(1 + t)$:

$$(1 + t) + 4(5 - 4t) = 3(5 - 4t)(1 + t)$$

$$1 + t + 20 - 16t = 3(5 + 5t - 4t - 4t^2)$$

$$21 - 15t = 3(5 + t - 4t^2)$$

$$21 - 15t = 15 + 3t - 12t^2$$

$$12t^2 - 18t + 6 = 0$$

Делим на 6:

$$2t^2 - 3t + 1 = 0$$

$$D = 9 - 8 = 1$$

$$t_{1,2} = \frac{3 \pm 1}{4}$$

$$t_1 = 1, \quad t_2 = \frac{1}{2}$$

**Возвращаемся к $x$:**  
$t = 1 \quad\Rightarrow\quad x = 2^1 = 2$  
$t = \frac{1}{2} \quad\Rightarrow\quad x = 2^{1/2} = \sqrt{2}$

**Проверка ОДЗ:**  
$x = 2$: $2 > 0$, $2 \neq 2^{5/4}$, $2 \neq 0.5$ ✅  
$x = \sqrt{2}$: $\sqrt{2} > 0$, $\sqrt{2} \neq 2^{5/4}$, $\sqrt{2} \neq 0.5$ ✅

---

**Ответ:** $x = 2, \sqrt{2}$

</LessonSection>
