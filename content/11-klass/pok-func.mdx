<LessonSection title="Определение и основные свойства">

Показательная функция имеет вид $ y = a^x $, где $ a > 0 $ и $ a \neq 1 $.

**Основные свойства:**
1. **Область определения:** все действительные числа $ (-\infty; +\infty) $
2. **Множество значений:** все положительные числа $ (0; +\infty) $
3. **Нули функции:** нет (так как $ a^x > 0 $ для любого $ x $)
4. **Пересечение с осью Oy:** точка $ (0; 1) $
5. **Промежутки знакопостоянства:** функция положительна на всей области определения
6. **Промежутки монотонности:**
   - При $ a > 1 $: функция возрастает на всей области определения
   - При $ 0 < a < 1 $: функция убывает на всей области определения

</LessonSection>

<LessonSection title="Графики показательной функции">

**График возрастающей функции ($ a > 1 $):**
- Проходит через точку $ (0; 1) $
- При $ x \to -\infty $ стремится к 0
- При $ x \to +\infty $ стремится к $ +\infty $

**График убывающей функции ($ 0 < a < 1 $):**
- Проходит через точку $ (0; 1) $
- При $ x \to -\infty $ стремится к $ +\infty $
- При $ x \to +\infty $ стремится к 0

</LessonSection>

<LessonSection title="Сравнение значений функции">

**Пример 1:**
Сравните $ f(3.5) $ и $ f(4.2) $, если $ f(x) = 0.3^x $

**Решение:**
1. Основание $ 0.3 < 1 $, значит функция убывает
2. Так как $ 3.5 < 4.2 $, то $ f(3.5) > f(4.2) $

**Ответ:** $ f(3.5) > f(4.2) $

**Пример 2:**
Сравните $ g(10.8) $ и $ g(12.9) $, если $ g(x) = (\sqrt{13})^x $

**Решение:**
1. Основание $ \sqrt{13} \approx 3.606 > 1 $, значит функция возрастает
2. Так как $ 10.8 < 12.9 $, то $ g(10.8) < g(12.9) $

**Ответ:** $ g(10.8) < g(12.9) $

<LessonSection title="Множество значений на промежутке">

**Пример 1:**
Найдите множество значений функции $ y = 2^x $ на промежутке $[-1; 5]$

**Решение:**
1. Функция возрастает (так как $ 2 > 1 $)
2. Найдем значения на концах:
   - $ x = -1 $: $ y = 2^{-1} = 0.5 $
   - $ x = 5 $: $ y = 2^5 = 32 $
3. Для возрастающей функции: $ E(y) = [0.5; 32] $

**Ответ:** $ [0.5; 32] $

**Пример 2:**
Найдите множество значений функции $ y = \left( \frac{1}{3} \right)^x $ на промежутке $[-2; 3]$

**Решение:**
1. Функция убывает (так как $ 0 < \frac{1}{3} < 1 $)
2. Найдем значения на концах:
   - $ x = -2 $: $ y = \left( \frac{1}{3} \right)^{-2} = 9 $
   - $ x = 3 $: $ y = \left( \frac{1}{3} \right)^3 = \frac{1}{27} $
3. Для убывающей функции: $ E(y) = \left[\frac{1}{27}; 9\right] $

**Ответ:** $ \left[\frac{1}{27}; 9\right] $


</LessonSection>

<LessonSection title="Наибольшее значение сложной функции">

**Пример:**
Найдите наибольшее значение функции $ y = (\sqrt{3})^{2 - x^2 + 4x} $

**Решение:**
1. Основание $ \sqrt{3} > 1 $, значит функция $ y = (\sqrt{3})^t $ возрастает
2. Наибольшее значение $ y $ будет при наибольшем значении показателя $ t = 2 - x^2 + 4x $
3. Найдем вершину параболы $ t(x) = -x^2 + 4x + 2 $:
   - $ x_0 = -\frac{4}{2 \cdot (-1)} = 2 $
   - $ t(2) = -4 + 8 + 2 = 6 $
4. Наибольшее значение исходной функции:
   $$ y_{\max} = (\sqrt{3})^6 = (3^{1/2})^6 = 3^3 = 27 $$

**Ответ:** 27


</LessonSection>

<LessonSection title="Ключевые выводы">

1. **Монотонность** зависит от основания:
   - $ a > 1 $ → функция возрастает
   - $ 0 < a < 1 $ → функция убывает

2. **При сравнении значений:**
   - Для возрастающей функции: чем больше $ x $, тем больше $ y $
   - Для убывающей функции: чем больше $ x $, тем меньше $ y $

3. **Нахождение множества значений на отрезке:**
   - Для возрастающей функции: $ E = [f(a); f(b)] $, где $ a < b $
   - Для убывающей функции: $ E = [f(b); f(a)] $, где $ a < b $

4. **Решение сложных задач:**
   - Сначала исследуйте показатель степени
   - Учитывайте монотонность показательной функции

</LessonSection>
