# Разношерстные функции

---

<LessonSection title="Обратная пропорциональность: функция y = k/x">

**Функция обратной пропорциональности:**

$$
y = \frac{k}{x}, \quad k \neq 0
$$

### **Свойства функции y = k/x**

**Область определения**

$$
D(f) = (-\infty; 0) \cup (0; +\infty)
$$

*Функция не определена при x = 0 (деление на ноль)*

---

**Множество значений**

$$
E(f) = (-\infty; 0) \cup (0; +\infty)
$$

*Функция никогда не принимает значение 0*

---

**Нули функции**

Нет нулей

*Уравнение k/x = 0 не имеет решений (k ≠ 0)*

---

**Промежутки знакопостоянства**
* При k > 0: y > 0 при x > 0; y < 0 при x < 0
* При k < 0: y < 0 при x > 0; y > 0 при x < 0

---

**График функции**

**Гипербола**, состоящая из двух ветвей

![](https://i.postimg.cc/2871XQKd/изображение.png)

*Асимптоты: x = 0 и y = 0*

---

**Промежутки монотонности**
* При k > 0: убывает на (-∞; 0) и на (0; +∞)
* При k < 0: возрастает на (-∞; 0) и на (0; +∞)

*Функция не является монотонной на всей области определения*

</LessonSection>

<LessonSection title="Модуль: функция y = |x|">

**Функция модуля:**

$$
y = |x| = 
\begin{cases} 
x, & \text{если } x \geq 0 \\
-x, & \text{если } x < 0 
\end{cases}
$$

### **Свойства функции y = |x|**

**Область определения**

$$
D(f) = (-\infty; +\infty)
$$

---

**Множество значений**

$$
E(f) = [0; +\infty)
$$

*Модуль всегда неотрицателен*

---

**Нули функции**

$$
x_0 = 0
$$

*|0| = 0*

---

**Промежутки знакопостоянства**
* y > 0 при x ≠ 0
* y = 0 при x = 0

---

**График функции**

**Ломаная линия**, состоящая из двух лучей

![](https://i.postimg.cc/bYHYzz9t/изображение.png)

*График симметричен относительно оси OY*

---

**Промежутки монотонности**
* Убывает на (-∞; 0]
* Возрастает на [0; +∞)

*x = 0 - точка минимума*

</LessonSection>

<LessonSection title="Степенная функция: y = x³ (кубическая функция)">

**Кубическая функция:**

$$
y = x^3
$$

### **Свойства функции y = x³**

**Область определения**

$$
D(f) = (-\infty; +\infty)
$$

*Функция определена для всех действительных чисел*

---

**Множество значений**

$$
E(f) = (-\infty; +\infty)
$$

*Функция принимает все действительные значения*

---

**Нули функции**

$$
x_0 = 0
$$

*x³ = 0 при x = 0*

---

**Промежутки знакопостоянства**
* y > 0 при x > 0
* y < 0 при x < 0
* y = 0 при x = 0

---

**График функции**

**Кубическая парабола**, симметричная относительно начала координат

![](https://i.postimg.cc/PfyTQ292/izobrazenie.png)

*Проходит через точки (-2, -8), (-1, -1), (0, 0), (1, 1), (2, 8)*

---

**Промежутки монотонности**

Возрастает на всей области определения

*Функция является строго возрастающей на (-∞; +∞)*

</LessonSection>

---

<LessonSection title="Степенная функция: y = √x (квадратный корень)">

**Функция квадратного корня:**

$$
y = \sqrt{x} \quad \text{или} \quad y = x^{\frac{1}{2}}
$$

### **Свойства функции y = √x**

**Область определения**

$$
D(f) = [0; +\infty)
$$

*Квадратный корень определен только для неотрицательных чисел*

---

**Множество значений**

$$
E(f) = [0; +\infty)
$$

*Квадратный корень всегда неотрицателен*

---

**Нули функции**

$$
x_0 = 0
$$

*√0 = 0*

---

**Промежутки знакопостоянства**
* y > 0 при x > 0
* y = 0 при x = 0

*Функция всегда неотрицательна*

---

**График функции**

**Ветвь параболы**, лежащая в первой координатной четверти

![](https://i.postimg.cc/CKqY9Yxr/изображение.png)

*Проходит через точки (0, 0), (1, 1), (4, 2), (9, 3)*

---

**Промежутки монотонности**

Возрастает на всей области определения

*Функция является строго возрастающей на [0; +∞)*

</LessonSection>
