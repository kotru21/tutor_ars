# Разные функции

## Обратная пропорциональность: функция y = k/x

<Definition>
**Функция обратной пропорциональности:**

$$
y = \frac{k}{x}, \quad k \neq 0
$$

</Definition>

### Свойства функции y = k/x

<LessonSection>
#### Область определения

$$
D(f) = (-\infty; 0) \cup (0; +\infty)
$$

<Explanation>Функция не определена при x = 0 (деление на ноль)</Explanation>

#### Множество значений

$$
E(f) = (-\infty; 0) \cup (0; +\infty)
$$

<Explanation>Функция никогда не принимает значение 0</Explanation>

#### Нули функции

$$
\text{Нет нулей}
$$

<Explanation>Уравнение k/x = 0 не имеет решений (k ≠ 0)</Explanation>

#### Промежутки знакопостоянства

- При k > 0: y > 0 при x > 0; y < 0 при x < 0
- При k < 0: y < 0 при x > 0; y > 0 при x < 0

#### График функции

**Гипербола**, состоящая из двух ветвей

![График функции y = k/x](https://i.postimg.cc/2871XQKd/изображение.png)

<Explanation>Асимптоты: x = 0 и y = 0</Explanation>

#### Промежутки монотонности

- При k > 0: убывает на (-∞; 0) и на (0; +∞)
- При k < 0: возрастает на (-∞; 0) и на (0; +∞)

<Explanation>
Функция не является монотонной на всей области определения
</Explanation>
</LessonSection>

## Модульная функция: y = |x|

<Definition>
**Функция модуля:**

$$
y = |x| = \begin{cases}
x, & \text{если } x \geq 0 \\
-x, & \text{если } x < 0
\end{cases}
$$

</Definition>

### Свойства функции y = |x|

<LessonSection>
#### Область определения

$$
D(f) = (-\infty; +\infty)
$$

#### Множество значений

$$
E(f) = [0; +\infty)
$$

<Explanation>Модуль всегда неотрицателен</Explanation>

#### Нули функции

$$
x_0 = 0
$$

<Explanation>|0| = 0</Explanation>

#### Промежутки знакопостоянства

- y > 0 при x ≠ 0
- y = 0 при x = 0

#### График функции

**Ломаная линия**, состоящая из двух лучей

![График функции y = |x|](https://i.postimg.cc/bYHYzz9t/изображение.png)

<Explanation>График симметричен относительно оси OY</Explanation>

#### Промежутки монотонности

- Убывает на (-∞; 0]
- Возрастает на [0; +∞)

<Explanation>
x = 0 - точка минимума
</Explanation>
</LessonSection>

## Кубическая функция: y = x³

<Definition>
**Кубическая функция:**

$$
y = x^3
$$

</Definition>

### Свойства функции y = x³

<LessonSection>
#### Область определения

$$
D(f) = (-\infty; +\infty)
$$

<Explanation>Функция определена для всех действительных чисел</Explanation>

#### Множество значений

$$
E(f) = (-\infty; +\infty)
$$

<Explanation>Функция принимает все действительные значения</Explanation>

#### Нули функции

$$
x_0 = 0
$$

<Explanation>x³ = 0 при x = 0</Explanation>

#### Промежутки знакопостоянства

- y > 0 при x > 0
- y < 0 при x < 0
- y = 0 при x = 0

#### График функции

**Кубическая парабола**, симметричная относительно начала координат

![График функции y = x³](https://i.postimg.cc/L8s8Crfm/изображение.png)

<Explanation>
  Проходит через точки (-2, -8), (-1, -1), (0, 0), (1, 1), (2, 8)
</Explanation>

#### Промежутки монотонности

$$
\text{Возрастает на всей области определения}
$$

<Explanation>
Функция является строго возрастающей на (-∞; +∞)
</Explanation>
</LessonSection>

## Функция квадратного корня: y = √x

<Definition>
**Функция квадратного корня:**

$$
y = \sqrt{x} \quad \text{или} \quad y = x^{\frac{1}{2}}
$$

</Definition>

### Свойства функции y = √x

<LessonSection>
#### Область определения

$$
D(f) = [0; +\infty)
$$

<Explanation>
  Квадратный корень определен только для неотрицательных чисел
</Explanation>

#### Множество значений

$$
E(f) = [0; +\infty)
$$

<Explanation>Квадратный корень всегда неотрицателен</Explanation>

#### Нули функции

$$
x_0 = 0
$$

<Explanation>√0 = 0</Explanation>

#### Промежутки знакопостоянства

- y > 0 при x > 0
- y = 0 при x = 0

<Explanation>Функция всегда неотрицательна</Explanation>

#### График функции

**Ветвь параболы**, лежащая в первой координатной четверти

![График функции y = √x](https://i.postimg.cc/CKqY9Yxr/изображение.png)

<Explanation>Проходит через точки (0, 0), (1, 1), (4, 2), (9, 3)</Explanation>

#### Промежутки монотонности

$$
\text{Возрастает на всей области определения}
$$

<Explanation>
Функция является строго возрастающей на [0; +∞)
</Explanation>
</LessonSection>
