<Section id="definition" title="Квадратное уравнение">

<Def title="Определение">
**Квадратным уравнением** называется уравнение вида:

$$ax^2 + bx + c = 0$$

где $a$, $b$, $c$ — числа (коэффициенты), причём $a \neq 0$.

</Def>

<Def title="Виды квадратных уравнений">
  - **Полное** квадратное уравнение: $ax^2 + bx + c = 0$ (все коэффициенты $\neq
  0$) - **Неполное** квадратное уравнение: - $ax^2 + bx = 0$ (при $c = 0$) -
  $ax^2 + c = 0$ (при $b = 0$) - $ax^2 = 0$ (при $b = 0$ и $c = 0$) -
  **Приведённое** квадратное уравнение: $x^2 + px + q = 0$ (при $a = 1$)
</Def>

</Section>

<Section id="discriminant" title="Дискриминант и формула корней">

<Def title="Дискриминант">
**Дискриминант** квадратного уравнения $ax^2 + bx + c = 0$:

$$D = b^2 - 4ac$$

</Def>

<Def title="Формула корней">
Корни квадратного уравнения находятся по формуле:

$$x_{1,2} = \frac{-b \pm \sqrt{D}}{2a}$$

</Def>

<Explanation>
**Количество корней зависит от знака дискриминанта:**
- $D > 0$ — **два различных корня**
- $D = 0$ — **один корень** (два совпадающих): $x = -\frac{b}{2a}$
- $D < 0$ — **нет действительных корней**
</Explanation>

</Section>

<Section id="vieta" title="Теорема Виета">

<Def title="Теорема Виета">
Для приведённого квадратного уравнения $x^2 + px + q = 0$ с корнями $x_1$ и $x_2$:

$$x_1 + x_2 = -p \quad \text{(сумма корней)}$$
$$x_1 \cdot x_2 = q \quad \text{(произведение корней)}$$

</Def>

<Def title="Общий случай">
Для уравнения $ax^2 + bx + c = 0$:

$$x_1 + x_2 = -\frac{b}{a}$$
$$x_1 \cdot x_2 = \frac{c}{a}$$

</Def>

<Explanation>
  Теорема Виета позволяет: - Быстро проверить найденные корни - Находить корни
  «подбором» для простых уравнений - Составлять уравнения по заданным корням
</Explanation>

</Section>

<Section id="examples" title="Примеры">

<Ex title="Пример 1: Решение через дискриминант">
Решить: $x^2 - 5x + 6 = 0$

**Решение:**
$$a = 1, \quad b = -5, \quad c = 6$$
$$D = (-5)^2 - 4 \cdot 1 \cdot 6 = 25 - 24 = 1$$
$$x_{1,2} = \frac{5 \pm \sqrt{1}}{2} = \frac{5 \pm 1}{2}$$
$$x_1 = 3, \quad x_2 = 2$$

**Ответ:** $x_1 = 3$, $x_2 = 2$

</Ex>

<Ex title="Пример 2: Проверка по теореме Виета">
Проверим корни $x_1 = 3$ и $x_2 = 2$ для уравнения $x^2 - 5x + 6 = 0$:

- Сумма: $3 + 2 = 5 = -(-5) = -p$ ✓
- Произведение: $3 \cdot 2 = 6 = q$ ✓
 </Ex>

<Ex title="Пример 3: Неполное уравнение">
Решить: $2x^2 - 8 = 0$

**Решение:**
$$2x^2 = 8$$
$$x^2 = 4$$
$$x = \pm 2$$

**Ответ:** $x_1 = 2$, $x_2 = -2$

</Ex>

<Ex title="Пример 4: Отрицательный дискриминант" isHard>
Решить: $x^2 + 2x + 5 = 0$

**Решение:**
$$D = 2^2 - 4 \cdot 1 \cdot 5 = 4 - 20 = -16 < 0$$

**Ответ:** Нет действительных корней

</Ex>

</Section>
