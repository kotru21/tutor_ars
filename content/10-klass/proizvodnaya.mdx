# Производная функции

## Определение производной

<Definition>
**Производной функции** $f(x)$ в точке $x_0$ называется предел отношения приращения функции к приращению аргумента при стремлении приращения аргумента к нулю:

$$f'(x_0) = \lim_{\Delta x \to 0} \frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x}$$

**Обозначения:** $f'(x)$, $y'$, $\frac{df}{dx}$, $\frac{dy}{dx}$

**Физический смысл:** мгновенная скорость изменения функции

</Definition>

<Example>
**Вычисление производной функции $f(x) = x^2$ по определению:**

1. $f(x_0) = x_0^2$

2. $f(x_0 + \Delta x) = (x_0 + \Delta x)^2 = x_0^2 + 2x_0\Delta x + (\Delta x)^2$

3. $f(x_0 + \Delta x) - f(x_0) = 2x_0\Delta x + (\Delta x)^2$

4. $\frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x} = \frac{2x_0\Delta x + (\Delta x)^2}{\Delta x} = 2x_0 + \Delta x$

5. $f'(x_0) = \lim_{\Delta x \to 0}(2x_0 + \Delta x) = 2x_0$

6. **Ответ:** $f'(x) = 2x$
</Example>  

## Правила вычисления производных

<LessonSection title="1. Производная константы">

<Definition>
$$(c)' = 0, \quad \text{где } c = \text{const}$$
</Definition>

<Example>$(5)' = 0$</Example>

</LessonSection>

<LessonSection title="2. Производная степенной функции">

<Definition>$$(x^n)' = n \cdot x^{n - 1}$$</Definition>

<Example>
$(x^3)' = 3x^2$

$(\sqrt{x})' = (x^{1/2})' = \frac{1}{2}x^{-1/2} = \frac{1}{2\sqrt{x}}$

</Example>

</LessonSection>

<LessonSection title="3. Производная суммы (разности)">

<Definition>$$(u(x) \pm v(x))' = u'(x) \pm v'(x)$$</Definition>

<Example>$(x^3 + 2x^2 - 5x + 3)' = 3x^2 + 4x - 5$</Example>

</LessonSection>

<LessonSection title="4. Производная произведения">

<Definition>
  $$(u(x) \cdot v(x))' = u'(x) \cdot v(x) + u(x) \cdot v'(x)$$
</Definition>

<Example>
$(x \cdot (x^2 + 1))' = x' \cdot (x^2 + 1) + x \cdot (x^2 + 1)'$

$= 1 \cdot (x^2 + 1) + x \cdot 2x$

$= x^2 + 1 + 2x^2 = 3x^2 + 1$

</Example>

</LessonSection>

<LessonSection title="5. Производная частного">

<Definition>
$$\left(\frac{u(x)}{v(x)}\right)' = \frac{u'(x) \cdot v(x) - u(x) \cdot v'(x)}{[v(x)]^2}$$
</Definition>

<Example>
$$\left(\frac{x}{x+1}\right)' = \frac{x' \cdot (x+1) - x \cdot (x+1)'}{(x+1)^2}$$

$$= \frac{1 \cdot (x+1) - x \cdot 1}{(x+1)^2} = \frac{x+1-x}{(x+1)^2} = \frac{1}{(x+1)^2}$$

</Example>

</LessonSection>

## Геометрический смысл производной

<Definition>
**Теорема:** Производная функции $f(x)$ в точке $x_0$ равна угловому коэффициенту касательной к графику функции в точке $M_0(x_0, f(x_0))$:

$$f'(x_0) = k = \tg\alpha$$

где $\alpha$ — угол наклона касательной к положительному направлению оси OX

</Definition>

<Example>
  <Explanation>
    Касательная к графику функции в точке $M_0$. Угол $\alpha$ определяет наклон
    касательной.
 </Explanation>
</Example>

## Исследование функции на монотонность

<Definition>
**Теорема:**
1. Если $f'(x) > 0$ на интервале $(a, b)$, то $f(x)$ возрастает на этом интервале
2. Если $f'(x) < 0$ на интервале $(a, b)$, то $f(x)$ убывает на этом интервале
</Definition>

## Точки экстремума функции

<Definition>
  **Определения:** - **Точка экстремума:** точки минимума и максимума - **Точка
  максимума:** точка $x_0$, в которой $f'(x_0) = 0$ и производная меняет знак с
  "+" на "-" - **Точка минимума:** точка $x_0$, в которой $f'(x_0) = 0$ и
  производная меняет знак с "-" на "+" - **Экстремум:** максимум или минимум
  функции - **Максимум функции:** значение $f(x)$ в точке максимума - **Минимум
  функции:** значение $f(x)$ в точке минимума
</Definition>

<Example variant="hard">
**Важное замечание:**

Если функция не имеет производной в точке, но непрерывна в ней, эта точка может быть точкой экстремума.

$$f(x) = |x|, \quad x_0 = 0 \text{ — точка минимума}$$

<Explanation>
Функция $|x|$ имеет минимум в точке $x=0$, но производной в этой точке не существует (излом)
</Explanation>
</Example>

## Пример: Нахождение промежутков монотонности

<Example>
**Дано:** $f(x) = 1 + 3x^2 + \frac{x^3}{3} - \frac{x^4}{4}$

**Решение:**

1. $D(f) = \mathbb{R}$ (все действительные числа)

2. **Находим производную:**

$$f'(x) = 0 + 6x + x^2 - x^3 = -x^3 + x^2 + 6x$$

$$f'(x) = -x(x^2 - x - 6) = -x(x - 3)(x + 2)$$

3. **Находим нули производной:**

$$f'(x) = 0 \Rightarrow x = 0, \quad x = 3, \quad x = -2$$

4. **Исследуем знак производной:**

| Интервал     | $(-\infty, -2)$ | $(-2, 0)$ | $(0, 3)$ | $(3, +\infty)$ |
| ------------ | --------------- | --------- | -------- | -------------- |
| Знак $f'(x)$ | +               | -         | +        | -              |
| Монотонность | ↗               | ↘         | ↗        | ↘              |

5. **Ответ:**

- Функция возрастает на $(-\infty, -2)$ и $(0, 3)$
- Функция убывает на $(-2, 0)$ и $(3, +\infty)$
- Точки максимума: $x = -2$ и $x = 3$
- Точка минимума: $x = 0$
</Example>

## Алгоритм исследования функции

<Definition>
**Алгоритм нахождения экстремумов:**

1. Найти область определения функции $D(f)$

2. Вычислить производную $f'(x)$

3. Найти критические точки (где $f'(x) = 0$ или не существует)

4. Определить знак производной на каждом интервале

5. По смене знака определить тип экстремума:
   - Знак меняется с "+" на "-" → точка максимума
   - Знак меняется с "-" на "+" → точка минимума

6. Вычислить значения функции в точках экстремума
  </Definition>
