Решение квадратичных неравенств

Общий алгоритм решения неравенства

Алгоритм решения квадратичного неравенства:

  1. Привести неравенство к стандартному виду: ax2+bx+c≤0 (или другие знаки неравенства)
  2. Найти корни квадратного уравнения ax2+bx+c=0
  3. Отметить корни на числовой прямой
  4. Определить направление ветвей параболы (знак коэффициента a)
  5. Схематически изобразить параболу, проходящую через отмеченные точки
  6. Выбрать интервалы, соответствующие знаку неравенства
  7. Записать ответ с учётом строгости неравенства

Пример 1: Решение неравенства

Графическое решение неравенства:
2 x 2 - x - 15 ≤ 0

Анализ решения:

  • Коэффициент a=2>0 ⇒ ветви параболы направлены вверх
  • Корни уравнения 2x2-x-15=0: x1=-2.5, x2=3
  • Неравенство нестрогое (≤) ⇒ точки, где функция равна нулю, включаем в ответ
  • Нам нужны промежутки, где парабола находится ниже или на оси OX

Ответ:

x ∈ [ - 2.5 ; 3 ]

Пример 2: Решение неравенства (методом умножения на -1)

Алгебраическое решение неравенства:
- 3 x 2 + x + 4 < 0

Шаг 1: Умножение на -1

Умножим обе части неравенства на -1. При умножении на отрицательное число знак неравенства меняется на противоположный:

( - 1 ) · ( - 3 x 2 + x + 4 ) > ( - 1 ) · 0
3 x 2 - x - 4 > 0

Шаг 2: Находим корни квадратного уравнения

Решаем уравнение 3x2-x-4=0:

  • Дискриминант: D = ( - 1 ) 2 - 4 · 3 · ( - 4 ) = 1 + 48 = 49
  • Корни: x 1 = 1 - 7 6 = - 1 , x 2 = 1 + 7 6 = 4 3

Шаг 3: Анализ неравенства

  • Коэффициент a=3>0 ⇒ ветви параболы направлены вверх.
  • Нам нужны значения x, при которых квадратный трёхчлен больше нуля.
  • Парабола с ветвями вверх больше нуля вне промежутка между корнями.

Шаг 4: Запись решения

Так как неравенство строгое (>), корни не включаются в решение. Получаем объединение двух интервалов:

Ответ (для неравенства после умножения на -1):

x ∈ ( - ∞ ; - 1 ) ∪ ( 4 3 ; + ∞ )

Пример 3а: Решение неравенства

Графическое решение неравенства:
x 2 + 3 > 0

Анализ решения:

  • Коэффициент a=1>0 ⇒ ветви параболы направлены вверх
  • Уравнение x2+3=0 не имеет действительных корней (дискриминант D=-12<0)
  • Парабола целиком находится выше оси OX
  • Неравенство строгое (>) ⇒ функция положительна для всех значений x

Ответ:

x ∈ ( - ∞ ; + ∞ )

Пример 3б: Решение неравенства

Графическое решение неравенства:
4 x 2 - 12 x + 9 > 0

Анализ решения:

  • Коэффициент a=4>0 ⇒ ветви параболы направлены вверх
  • Уравнение 4x2-12x+9=0 имеет один корень (дискриминант D=0): x0=32
  • Парабола касается оси OX в точке x=32 и положительна во всех остальных точках
  • Неравенство строгое (>) ⇒ точка касания не включается в ответ

Ответ:

x ∈ ( - ∞ ; 3 2 ) ∪ ( 3 2 ; + ∞ )

Решение систем и совокупностей квадратичных неравенств

Решение систем квадратичных неравенств

Алгоритм решения системы квадратичных неравенств:

  1. Решить каждое неравенство системы отдельно
  2. Найти множество решений для каждого неравенства
  3. Найти пересечение полученных множеств решений
  4. Изобразить решение на числовой прямой
  5. Записать окончательный ответ

Пример 1: Решение системы неравенств

{ 4 x 2 ≥ - ( x - 3 ) x 2 ≤ 6

Анализ решения:

  • Первое неравенство: 4x2≥-x+3 или 4x2+x-3≥0

  • Решение первого неравенства: x∈(-∞;-1]∪[34;+∞)
  • Второе неравенство: x2≤6
  • Решение второго неравенства: x∈[-6;6]
  • Пересечение решений: объединяем решения обоих неравенств

Ответ:

x ∈ [ - 6 ; - 1 ] ∪ [ 3 4 ; 6 ]

Решение совокупностей квадратичных неравенств

Алгоритм решения совокупности квадратичных неравенств:

  1. Решить каждое неравенство совокупности отдельно
  2. Найти множество решений для каждого неравенства
  3. Найти объединение полученных множеств решений
  4. Изобразить решение на числовой прямой
  5. Записать окончательный ответ

Пример 2: Решение совокупности неравенств

[ 3 x 2 ≤ 9 4 x 2 > 2.56

Анализ решения:

  • Первое неравенство: 3x2≤9 или x2≤3
  • Решение первого неравенства: x∈[-3;3]
  • Второе неравенство: 4x2>2.56 или x2>0.64
  • Решение второго неравенства: x∈(-∞;-0.8)∪(0.8;+∞)
  • Объединение решений: берем все значения, удовлетворяющие хотя бы одному неравенству
Графическое решение совокупности:

Ответ:

x ∈ ( - ∞ ; + ∞ )

Поскольку первое неравенство дает промежуток [-3;3], а второе неравенство покрывает все значения вне промежутка [-0.8;0.8], их объединение дает всю числовую прямую.
