Введение в квадратные уравнения. Неполные квадратные уравнения

Что такое квадратное уравнение?

Квадратным уравнением называется уравнение вида:

a x 2 + b x + c = 0

где:

  • x — переменная (неизвестное)
  • a, b, c — некоторые числа ( коэффициенты)
  • a ≠ 0 (если a = 0, уравнение становится линейным)

Пояснение: Квадратное уравнение называется так потому, что наивысшая степень переменной x — вторая (x²). Число a называется первым коэффициентом или коэффициентом при x², число b — вторым коэффициентом или коэффициентом при x, а число c — свободным членом.

Неполные квадратные уравнения

Неполным квадратным уравнением называется квадратное уравнение, в котором один или два коэффициента (кроме a) равны нулю.

Существует три вида неполных квадратных уравнений:

Вид 1: Уравнение без свободного члена (c = 0)
a x 2 + b x = 0

где a ≠ 0 и b ≠ 0.

Вид 2: Уравнение без линейного члена (b = 0)
a x 2 + c = 0

где a ≠ 0 и c ≠ 0.

Вид 3: Уравнение только с квадратом переменной
a x 2 = 0

где a ≠ 0 (b = 0 и c = 0).

Решение неполных квадратных уравнений

Пример 1: Решение уравнения вида ax² = 0

Решите уравнение: − 2 x 2 = 0

Решение:

1. Разделим обе части уравнения на -2:

− 2 x 2 − 2 = 0 − 2

2. Упрощаем:

x 2 = 0

3. Извлекаем квадратный корень:

x = 0

Ответ: x = 0

Пример 2: Решение уравнения вида ax² + bx = 0

Решите уравнение: 5 x 2 + 2 x = 0

Решение:

1. Вынесем общий множитель x за скобки:

x ( 5 x + 2 ) = 0

2. Произведение равно нулю, когда хотя бы один из множителей равен нулю:

x = 0  или  5 x + 2 = 0

3. Решаем второе уравнение:

5 x = − 2
x = − 2 5

Ответ: x 1 = 0 ,   x 2 = − 2 5

Пример 3: Решение уравнения вида ax² + c = 0 (с > 0)

Решите уравнение: x 2 − 3 = 0

Решение:

1. Перенесём свободный член в правую часть:

x 2 = 3

2. Извлечём квадратный корень из обеих частей:

x = ± 3

Ответ: x = ± 3

Пример 4: Решение уравнения вида ax² + c = 0 (c < 0, a < 0)

Решите уравнение: − x 2 − 1 = 0

Решение:

1. Перенесём свободный член в правую часть:

− x 2 = 1

2. Разделим обе части на -1:

x 2 = − 1

3. Квадрат действительного числа не может быть отрицательным

Ответ: уравнение не имеет действительных корней

Решение полных квадратных уравнений.

Что такое полное квадратное уравнение?

Полным квадратным уравнением называется уравнение вида:

a x 2 + b x + c = 0

где:

  • a, b, c — коэффициенты, причём a ≠ 0
  • Все три коэффициента отличны от нуля: a ≠ 0, b ≠ 0, c ≠ 0

Пояснение: В отличие от неполных квадратных уравнений, в полных квадратных уравнениях присутствуют все три члена: квадратичный член (ax²), линейный член (bx) и свободный член (c).

Дискриминант квадратного уравнения

Дискриминантом квадратного уравнения называется выражение:

D = b 2 − 4 a c

Дискриминант (от латинского "discriminare" — различать) определяет количество и тип корней квадратного уравнения.

Правило определения корней по дискриминанту
Дискриминант D Количество корней Тип корней
D > 0 Два различных корня Действительные, различные
D = 0 Один корень (два одинаковых) Действительные, равные
D < 0 Нет действительных корней Комплексные (в школьном курсе — "нет корней")

Формула корней квадратного уравнения

Формула для нахождения корней квадратного уравнения:

x 1,2 = − b ± D 2 a

где D = b² - 4ac — дискриминант.

Пояснение: Знак ± означает, что нужно вычислить два значения: одно со знаком плюс, другое со знаком минус. Формула работает для любого квадратного уравнения с действительными коэффициентами.

Примеры решения полных квадратных уравнений

Пример 1: Уравнение с D > 0 (два различных корня)

Решите уравнение: 2 x 2 − 5 x + 2 = 0

Решение:

1. Определяем коэффициенты: a = 2, b = -5, c = 2

2. Вычисляем дискриминант:

D = b 2 − 4 a c = ( − 5 ) 2 − 4 · 2 · 2 = 25 − 16 = 9

3. Так как D = 9 > 0, уравнение имеет два различных корня

4. Находим корни по формуле:

x 1,2 = − b ± D 2 a = 5 ± 9 2 · 2 = 5 ± 3 4

5. Вычисляем два корня:

x 1 = 5 + 3 4 = 8 4 = 2
x 2 = 5 − 3 4 = 2 4 = 1 2

Ответ: x 1 = 2 ,   x 2 = 1 2

Пример 2: Уравнение с D = 0 (один корень)

Решите уравнение: x 2 − 6 x + 9 = 0

Решение:

1. Определяем коэффициенты: a = 1, b = -6, c = 9

2. Вычисляем дискриминант:

D = b 2 − 4 a c = ( − 6 ) 2 − 4 · 1 · 9 = 36 − 36 = 0

3. Так как D = 0, уравнение имеет один корень (два одинаковых)

4. Находим корень по формуле:

x = − b 2 a = 6 2 · 1 = 6 2 = 3

Ответ: x = 3

Пример 3: Уравнение с D < 0 (нет действительных корней)

Решите уравнение: 3 x 2 − 4 x + 5 = 0

Решение:

1. Определяем коэффициенты: a = 3, b = -4, c = 5

2. Вычисляем дискриминант:

D = b 2 − 4 a c = ( − 4 ) 2 − 4 · 3 · 5 = 16 − 60 = − 44

3. Так как D = -44 < 0, уравнение не имеет действительных корней

Ответ: уравнение не имеет действительных корней

Решение приведенных квадратных уравнений.

Теорема Виета для приведённого квадратного уравнения

Теорема Виета устанавливает связь между корнями квадратного уравнения и его коэффициентами.

Для приведённого квадратного уравнения вида:

x 2 + p x + q = 0

верны следующие соотношения:

x 1 + x 2 = − p
x 1 · x 2 = q

где x₁ и x₂ — корни уравнения.

Пояснение: Теорема Виета позволяет:

  • Находить корни уравнения подбором
  • Проверять правильность найденных корней
  • Находить выражения, составленные из корней, без вычисления самих корней

Теорема Виета для общего квадратного уравнения

Для общего квадратного уравнения вида:

a x 2 + b x + c = 0  ( a ≠ 0 )

соотношения Виета имеют вид:

x 1 + x 2 = − b a
x 1 · x 2 = c a

Примеры решения уравнений с помощью теоремы Виета

Пример 1: Решение уравнения подбором

Решите уравнение: x 2 − 4 x + 3 = 0

Решение с использованием теоремы Виета:

1. Запишем условия теоремы Виета для данного уравнения:

x 1 + x 2 = 4  (так как -p = -(-4) = 4)
x 1 · x 2 = 3

2. Подбираем пару чисел, сумма которых равна 4, а произведение равно 3:

Проверяем возможные варианты:

  • 1 + 3 = 4 и 1 × 3 = 3 ✓
  • 2 + 2 = 4, но 2 × 2 = 4 ≠ 3
  • 4 + 0 = 4, но 4 × 0 = 0 ≠ 3

Ответ: x 1 = 1 ,   x 2 = 3

Пример 2: Решение уравнения подбором

Решите уравнение: x 2 + 7 x + 10 = 0

Решение с использованием теоремы Виета:

1. Запишем условия теоремы Виета для данного уравнения:

x 1 + x 2 = − 7
x 1 · x 2 = 10

2. Подбираем пару чисел, сумма которых равна -7, а произведение равно 10:

Оба корня должны быть отрицательными (так как сумма отрицательна, а произведение положительно).

Проверяем возможные варианты:

  • (-2) + (-5) = -7 и (-2) × (-5) = 10 ✓
  • (-1) + (-6) = -7, но (-1) × (-6) = 6 ≠ 10
  • (-3) + (-4) = -7, но (-3) × (-4) = 12 ≠ 10

Ответ: x 1 = − 2 ,   x 2 = − 5

Выражения, составленные из корней

С помощью теоремы Виета можно вычислять различные выражения, составленные из корней, не находя сами корни.

Основные полезные формулы:

  1. Сумма квадратов корней:
    x 1 2 + x 2 2 = ( x 1 + x 2 ) 2 − 2 x 1 x 2
  2. Сумма кубов корней:
    x 1 3 + x 2 3 = ( x 1 + x 2 ) 3 − 3 x 1 x 2 ( x 1 + x 2 )
  3. Выражение вида x₁²x₂ + x₁x₂²:
    x 1 2 x 2 + x 1 x 2 2 = x 1 x 2 ( x 1 + x 2 )

Пример вычисления выражений без нахождения корней

Задача

Известно, что x 1 и x 2 — корни уравнения x 2 + 7 x − 12 = 0 . Не решая уравнения, найдите значение выражения:

Решение:

1. Сначала определим по теореме Виета:

x 1 + x 2 = − 7
x 1 · x 2 = − 12

а) Найдём ( x 1 + x 2 ) 2 :

( x 1 + x 2 ) 2 = ( − 7 ) 2 = 49

б) Найдём x 1 2 x 2 + x 1 x 2 2 :

x 1 2 x 2 + x 1 x 2 2 = x 1 x 2 ( x 1 + x 2 ) = ( − 12 ) · ( − 7 ) = 84

в) Найдём x 1 2 + x 2 2 :

x 1 2 + x 2 2 = ( x 1 + x 2 ) 2 − 2 x 1 x 2 = ( − 7 ) 2 − 2 · ( − 12 ) = 49 + 24 = 73

Ответ: а) 49, б) 84, в) 73

Обратная теорема Виета

Обратная теорема Виета: Если числа x1 и x2 таковы, что

x 1 + x 2 = − p
x 1 · x 2 = q

то эти числа являются корнями квадратного уравнения:

x 2 + p x + q = 0
Пример использования обратной теоремы

Составьте квадратное уравнение, корни которого равны 2 и -5.

Решение:

1. Находим сумму корней: 2 + (-5) = -3

2. Находим произведение корней: 2 × (-5) = -10

3. По обратной теореме Виета, уравнение имеет вид:

x 2 + 3 x − 10 = 0

Ответ: x 2 + 3 x − 10 = 0

Разложение на множители

Теорема о разложении квадратного трехчлена на множители

Квадратный трехчлен — это выражение вида:

a x 2 + b x + c

где a ≠ 0 .

Теорема: Если квадратное уравнение a x 2 + b x + c = 0 имеет корни x 1 и x 2 , то квадратный трехчлен можно разложить на множители:

a x 2 + b x + c = a ( x − x 1 ) ( x − x 2 )

Пояснение: Эта теорема позволяет разложить квадратный трехчлен на линейные множители, если соответствующее квадратное уравнение имеет корни. Если уравнение не имеет корней ( D < 0 ), то разложение на линейные множители с действительными коэффициентами невозможно.

Алгоритм разложения квадратного трехчлена на множители

Шаги разложения:

  1. Найти корни квадратного уравнения a x 2 + b x + c = 0
  2. Если уравнение имеет два корня x 1 и x 2 , то разложить по формуле: a ( x − x 1 ) ( x − x 2 )
  3. Если уравнение имеет один корень x 1 ( D = 0 ), то: a ( x − x 1 ) 2
  4. Если уравнение не имеет корней ( D < 0 ), то разложение на линейные множители с действительными коэффициентами невозможно

Примеры разложения квадратных трехчленов на множители

Пример 1: 4 x 2 − 8 x + 3

Решение:

1. Составляем квадратное уравнение: 4 x 2 − 8 x + 3 = 0

2. Находим дискриминант:

D = b 2 − 4 a c = ( − 8 ) 2 − 4 · 4 · 3 = 64 − 48 = 16

3. Находим корни:

x 1,2 = 8 ± 16 2 · 4 = 8 ± 4 8

4. Вычисляем:

x 1 = 8 + 4 8 = 12 8 = 3 2 , x 2 = 8 − 4 8 = 4 8 = 1 2

5. Разлагаем на множители:

4 x 2 − 8 x + 3 = 4 ( x − 3 2 ) ( x − 1 2 )

6. Можно записать без дробей, умножив коэффициенты:

4 x 2 − 8 x + 3 = ( 2 x − 3 ) ( 2 x − 1 )

Ответ: ( 2 x − 3 ) ( 2 x − 1 )

Пример 2: 81 t 2 − 36 t + 4

Решение:

1. Составляем уравнение: 81 t 2 − 36 t + 4 = 0

2. Находим дискриминант:

D = ( − 36 ) 2 − 4 · 81 · 4 = 1296 − 1296 = 0

3. Так как D = 0, уравнение имеет один корень:

t = 36 2 · 81 = 36 162 = 2 9

4. Разлагаем на множители:

81 t 2 − 36 t + 4 = 81 ( t − 2 9 ) 2

5. Можно записать как полный квадрат:

81 t 2 − 36 t + 4 = ( 9 t − 2 ) 2

Ответ: ( 9 t − 2 ) 2

Пример 3: 8 t 2 − 6 t + 3

Решение:

1. Составляем уравнение: 8 t 2 − 6 t + 3 = 0

2. Находим дискриминант:

D = ( − 6 ) 2 − 4 · 8 · 3 = 36 − 96 = − 60

3. Так как D = -60 < 0, квадратное уравнение не имеет действительных корней

4. Следовательно, квадратный трехчлен нельзя разложить на линейные множители с действительными коэффициентами

Ответ: разложение на линейные множители с действительными коэффициентами невозможно

Приложение

Биквадратные уравнения

Биквадратным уравнением называется уравнение вида:

a x 4 + b x 2 + c = 0

где a ≠ 0 .

Пояснение: Биквадратное уравнение содержит только чётные степени переменной. Его можно свести к квадратному уравнению с помощью замены переменной.

Метод решения биквадратных уравнений

Алгоритм решения:

  1. Сделать замену переменной: t = x 2 , при этом t ≥ 0
  2. Получить квадратное уравнение: a t 2 + b t + c = 0
  3. Решить полученное квадратное уравнение
  4. Для каждого неотрицательного корня t ≥ 0 решить уравнение x 2 = t
  5. Записать все действительные корни исходного уравнения

Примеры решения биквадратных уравнений

Пример a: x 4 − 10 x 2 + 9 = 0

Решение:

1. Делаем замену: t = x 2 , тогда t ≥ 0

2. Получаем квадратное уравнение:

t 2 − 10 t + 9 = 0

3. Решаем квадратное уравнение:

D = ( − 10 ) 2 − 4 · 1 · 9 = 100 − 36 = 64
t 1,2 = 10 ± 64 2 = 10 ± 8 2

4. Находим значения t :

t 1 = 10 + 8 2 = 9 , t 2 = 10 − 8 2 = 1

5. Возвращаемся к переменной x :

  • Если t = 9 , то x 2 = 9 ⇒ x = ± 3
  • Если t = 1 , то x 2 = 1 ⇒ x = ± 1

Ответ: x 1 = − 3 , x 2 = − 1 , x 3 = 1 , x 4 = 3

Уравнения, сводящиеся к квадратным с помощью замены переменной

Некоторые уравнения более сложного вида можно свести к квадратным уравнениям с помощью замены переменной.

Пример: ( x 2 − 4 x ) 2 + 7 ( x 2 − 4 x ) + 12 = 0

Решение:

1. Замечаем повторяющееся выражение x 2 − 4 x

2. Делаем замену: t = x 2 − 4 x

3. Получаем квадратное уравнение:

t 2 + 7 t + 12 = 0

4. Решаем квадратное уравнение:

D = 7 2 − 4 · 1 · 12 = 49 − 48 = 1
t 1,2 = − 7 ± 1 2

5. Находим значения t :

t 1 = − 7 + 1 2 = − 3 , t 2 = − 7 − 1 2 = − 4

6. Возвращаемся к переменной x :

Случай 1: x 2 − 4 x = − 3

x 2 − 4 x + 3 = 0

Решаем: D = 16 − 12 = 4 , x = 4 ± 2 2

x 1 = 1 , x 2 = 3

Случай 2: x 2 − 4 x = − 4

x 2 − 4 x + 4 = 0

Решаем: ( x − 2 ) 2 = 0 ⇒ x = 2

Ответ: x 1 = 1 , x 2 = 2 , x 3 = 3

Уравнения с модулем, сводящиеся к квадратным

Уравнения, содержащие модуль, часто решаются рассмотрением случаев.

Пример: x 2 − 3 | x | + 2 = 0

Решение:

1. Рассмотрим два случая в зависимости от знака x

Случай 1: x ≥ 0 , тогда | x | = x

Уравнение принимает вид: x 2 − 3 x + 2 = 0

Решаем: D = 9 − 8 = 1 , x = 3 ± 1 2

x 1 = 1 , x 2 = 2 (оба удовлетворяют условию x ≥ 0 )

Случай 2: x < 0 , тогда | x | = − x

Уравнение принимает вид: x 2 + 3 x + 2 = 0

Решаем: D = 9 − 8 = 1 , x = − 3 ± 1 2

x 3 = − 1 , x 4 = − 2 (оба удовлетворяют условию x < 0 )

Ответ: x 1 = − 2 , x 2 = − 1 , x 3 = 1 , x 4 = 2

Альтернативный метод: Уравнение x 2 − 3 | x | + 2 = 0 можно решить с помощью замены t = | x | , t ≥ 0 . Тогда уравнение принимает вид t 2 − 3 t + 2 = 0 .

Другие типы уравнений, сводящихся к квадратным

1. Уравнения с дробными степенями:

Пример: x 2 / 3 − 5 x 1 / 3 + 6 = 0

Решение: Замена t = x 1 / 3 ⇒ t 2 − 5 t + 6 = 0

2. Уравнения с радикалами:

Пример: x − 3 x + 2 = 0

Решение: Замена t = x , t ≥ 0 ⇒ t 2 − 3 t + 2 = 0

3. Возвратные уравнения:

Пример: x 4 + x 3 − 4 x 2 + x + 1 = 0

Решение: Деление на x 2 и замена t = x + 1 x
