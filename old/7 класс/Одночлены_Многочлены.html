Одночлены. Многочлены

Одночлены: определение и основные понятия

Определение одночлена:

Одночленом называется выражение, которое представляет собой:

1) произведение чисел и переменных
2) натуральные степени переменных
3) отдельное число или переменная
Примеры одночленов:
5 x 2 y , - 3.7 a 3 , 2 5 m 4 n 2 , 12 , k
Ловушки: что НЕ является одночленом?
2 x + 3 y ← содержит сложение
4 x ← деление на переменную
x - 2 ← отрицательная степень
x ← корень (не натуральная степень)

Стандартный вид одночлена

Стандартный вид одночлена:

Одночлен записан в стандартном виде, если:

  1. На первом месте стоит числовой множитель (коэффициент)
  2. За ним записаны произведения степеней различных переменных
  3. Каждая переменная встречается только один раз
Приведение к стандартному виду:
4 x · 2 x y = 8 x 2 y
- 5 a 3 x · 4 c = - 20 a 3 x c
- 2 a 2 y 2 ( - 5 a 3 y 5 ) = 10 a 5 y 7

Коэффициент одночлена:

Числовой множитель одночлена, записанного в стандартном виде

- 15 a 3 x 4 c 3 → коэффициент = - 15 2 7 x 3 y → коэффициент = 2 7 a 5 y 7 → коэффициент = 1 - x 2 → коэффициент = - 1

Степень одночлена

Определение степени одночлена:

Степенью одночлена с коэффициентом, отличным от нуля, называется сумма показателей степеней всех переменных, входящих в этот одночлен.

Степень одночлена = ∑ всех переменных показатель степени
Примеры определения степени:
5 a 2 b 3 c → 2 + 3 + 1 = 6 (6-я степень)
10 a 5 y 7 → 5 + 7 = 12 (12-я степень)
- 4 x 5 → 5 (5-я степень)
25 x → 1 (1-я степень)
7 → 0 (0-я степень)
Важное правило:

Если одночлен не содержит переменных (это просто число), то его степень равна нулю.

1024 , - 3.7 , 2 5 → степень = 0

Действия с одночленами

Умножение одночленов

Чтобы умножить одночлены, нужно:

  1. Перемножить коэффициенты
  2. Перемножить степени с одинаковыми основаниями
2 x 2 y · ( 0.3 x 3 y 2 ) = ( 2 · 0.3 ) · ( x 2 · x 3 ) · ( y · y 2 ) = 0.6 x 5 y 3

Деление одночленов

Чтобы разделить одночлены, нужно:

  1. Разделить коэффициенты
  2. Разделить степени с одинаковыми основаниями
15 x 4 y 3 - 3 x y 2 = 15 - 3 · x 4 x · y 3 y 2 = - 5 x 3 y

Возведение одночлена в степень

Чтобы возвести одночлен в степень, нужно возвести в эту степень каждый множитель.

( 0.1 a 6 b 3 ) 2 = ( 0.1 ) 2 · ( a 6 ) 2 · ( b 3 ) 2 = 0.01 a 12 b 6

Подобные одночлены

Определение подобных одночленов:

Одночлены называются подобными, если они имеют одинаковую буквенную часть (одинаковые переменные в одинаковых степенях).

3 x 3 y и 5 x 3 y → подобные 5 a 4 b 3 и - 4 a 4 b 3 → подобные 2 x 2 y и 2 x y 2 → НЕ подобные

Сложение и вычитание подобных одночленов

Складывать и вычитать можно только подобные одночлены!

2 a 3 b 2 + 7 a 3 b 2 = ( 2 + 7 ) a 3 b 2 = 9 a 3 b 2
10 b 2 c 3 - 7 b 2 c 3 - 4 b 2 c 3 = ( 10 - 7 - 4 ) b 2 c 3 = - 1 b 2 c 3
Ловушка при сложении одночленов:

Нельзя складывать неподобные одночлены!

3 x 2 + 2 x → НЕЛЬЗЯ упростить!
5 a b + 3 a c → НЕЛЬЗЯ упростить!

Многочлены: определение и основные понятия

Определение многочлена:

Многочленом называется сумма одночленов.

Примеры многочленов:
3 x 3 - 2 x y 2 + y - 2
5 x 2 - 2 y 3 ( двучлен )

Стандартный вид многочлена:

Многочлен имеет стандартный вид, если:

  1. Все его члены записаны в стандартном виде
  2. Среди членов нет подобных
2 x 3 + 5 x x y - 7.5 x x y → 2 x 3 - 2.5 x 2 y

Степень многочлена:

Степенью многочлена стандартного вида называется наибольшая из степеней входящих в него одночленов.

- 3 x 5 y 4 + 3 x 5 y - 9 x 6 + 4 x 5 y - 5 x 5 y 4 → - 8 x 5 y 4 + 7 x 5 y - 9 x 6

Степень многочлена = 9 (максимум из 5+4=9, 5+1=6, 6=6)

Действия с многочленами

Сложение и вычитание многочленов

( - 3 x 2 y 2 + 2 x y - 7 ) + ( x 2 y 2 + 4 x y - 2 ) = - 2 x 2 y 2 + 6 x y - 9

Умножение одночлена на многочлен

Чтобы умножить одночлен на многочлен, нужно умножить одночлен на каждый член многочлена.

3 x y ( 4 x - 7 y ) = 12 x 2 y - 21 x y 2

Деление многочлена на одночлен

Чтобы разделить многочлен на одночлен, нужно разделить каждый член многочлена на этот одночлен.

( 4 x 4 - 6 x 3 + 8 x 2 ) : ( 2 x 2 ) = 2 x 2 - 3 x + 4

Умножение многочлена на многочлен

Чтобы умножить многочлен на многочлен, нужно каждый член одного многочлена умножить на каждый член другого многочлена.

( x + 3 ) ( 4 x - 2 ) = 4 x 2 - 2 x + 12 x - 6 = 4 x 2 + 10 x - 6

Формулы сокращенного умножения

Квадрат суммы

( a + b ) 2 = a 2 + 2 a b + b 2
Пример:
( x + 3 ) 2 = x 2 + 6 x + 9

Квадрат разности

( a - b ) 2 = a 2 - 2 a b + b 2
Пример:
( 7 n - 1 ) 2 = 49 n 2 - 14 n + 1

Разность квадратов

( a + b ) ( a - b ) = a 2 - b 2
Пример:
( 3 b + 7 c ) ( 3 b - 7 c ) = 9 b 2 - 49 c 2

Разложение многочленов на множители

Вынесение общего множителя за скобки

6 a 3 b 4 + 9 a 2 b 2 c = 3 a 2 b 2 ( 2 a b 2 + 3 c )

Способ группировки

a x + 7 a - 3 x - 21 = ( a x + 7 a ) + ( - 3 x - 21 )
= a ( x + 7 ) - 3 ( x + 7 ) = ( x + 7 ) ( a - 3 )

Использование формул сокращенного умножения

9 25 m 4 - n 6 = ( 3 5 m 2 ) 2 - ( n 3 ) 2
= ( 3 5 m 2 + n 3 ) ( 3 5 m 2 - n 3 )
