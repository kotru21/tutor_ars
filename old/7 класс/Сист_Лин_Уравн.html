Системы линейных уравнений

Системы линейных уравнений: основные понятия

Определение системы линейных уравнений:

Системой линейных уравнений называется набор из двух или более линейных уравнений с двумя или более переменными, которые рассматриваются совместно.

{ 2x+3y=7 x-y=1

Решение системы уравнений:

Решением системы уравнений называется упорядоченная пара чисел (для системы с двумя переменными), которая удовлетворяет всем уравнениям системы одновременно.

Пример проверки решения:
Для системы  { 2x+y=5 x-y=1

Проверим, является ли пара (2, 1) решением:

1) 2·2 + 1 = 5 ✓
2) 2 - 1 = 1 ✓

Оба уравнения выполняются, значит (2, 1) — решение системы.

Геометрическая интерпретация:

Каждое линейное уравнение с двумя переменными представляет собой прямую на координатной плоскости. Решение системы — это точка пересечения этих прямых.

Метод сложения (метод исключения)

Алгоритм метода сложения:

  1. Умножить уравнения на такие числа, чтобы коэффициенты при одной из переменных стали противоположными
  2. Сложить уравнения почленно
  3. Решить полученное уравнение с одной переменной
  4. Подставить найденное значение в любое из исходных уравнений
  5. Найти вторую переменную
  6. Записать ответ
Пример 1: Простая система

Решить систему:

{ 3x+2y=8 2x-2y=2

1. Складываем уравнения почленно (коэффициенты при y уже противоположны):

3x+2y+2x-2y=8+2

2. Упрощаем (y сокращаются):

5x=10

3. Находим x:

x=2

4. Подставляем x = 2 в первое уравнение:

3·2+2y=8

5. Решаем относительно y:

6+2y=8
2y=2
y=1
Ответ: (2; 1)
Пример 2: Умножение уравнений

Решить систему:

{ 2x+3y=7 3x+2y=8

1. Умножаем первое уравнение на 3, второе на -2:

{ 6x+9y=21 -6x-4y=-16

2. Складываем уравнения:

5y=5

3. Находим y:

y=1

4. Подставляем y = 1 в первое уравнение:

2x+3·1=7
2x=4
x=2
Ответ: (2; 1)

Метод подстановки

Алгоритм метода подстановки:

  1. Выразить одну переменную через другую из любого уравнения
  2. Подставить полученное выражение в другое уравнение
  3. Решить полученное уравнение с одной переменной
  4. Найти вторую переменную
  5. Записать ответ
Пример 1: Простая подстановка

Решить систему:

{ x+y=5 2x-y=1

1. Из первого уравнения выражаем y:

y=5-x

2. Подставляем во второе уравнение:

2x-(5-x)=1

3. Решаем уравнение:

2x-5+x=1
3x=6
x=2

4. Находим y:

y=5-2=3
Ответ: (2; 3)
Пример 2: Более сложная подстановка

Решить систему:

{ 2x-3y=1 3x+y=7

1. Из второго уравнения выражаем y:

y=7-3x

2. Подставляем в первое уравнение:

2x-3(7-3x)=1

3. Решаем уравнение:

2x-21+9x=1
11x=22
x=2

4. Находим y:

y=7-3·2=1
Ответ: (2; 1)

Количество решений системы линейных уравнений

Три возможных случая:

Система двух линейных уравнений с двумя переменными может иметь:

  1. Единственное решение
  2. Бесконечно много решений
  3. Не иметь решений
Случай 1: Единственное решение

Прямые пересекаются в одной точке.

{ 2x+y=5 x-y=1

Решение: (2; 1) — единственная точка пересечения.

Случай 2: Бесконечно много решений

Прямые совпадают.

{ 2x+2y=4 x+y=2

Второе уравнение — это первое, деленное на 2. Все точки прямой x + y = 2 являются решениями.

Ответ: Бесконечно много решений.

Случай 3: Нет решений

Прямые параллельны и не совпадают.

{ 2x+2y=4 2x+2y=6

Левые части одинаковы, а правые разные. При вычитании получаем 0 = -2 — противоречие.

Ответ: Нет решений.

Как определить количество решений:

{ a1x+b1y=c1 a2x+b2y=c2

1. Единственное решение: если a1a2≠b1b2

2. Бесконечно много решений: если a1a2=b1b2=c1c2

3. Нет решений: если a1a2=b1b2≠c1c2

Важные замечания:

1. Оба метода дают одинаковый результат при правильном применении.

2. Выбор метода зависит от конкретной системы.

3. Всегда проверяйте решение подстановкой в оба уравнения.

4. В случае противоречия (например, 0 = 5) система не имеет решений.

5. В случае тождества (например, 0 = 0) система имеет бесконечно много решений.
