Корень n-й степени и его свойства

Определение

Корень n-ой степени

Определение: Корнем n-ой степени из числа a называется такое число b, что bⁿ = a.

an = b такое, что bn = a

Обозначения:

  • a — подкоренное выражение
  • n — показатель корня (степень корня)
  • b — значение корня

Особые случаи:

  • a2 — квадратный корень (обычно пишут √a)
  • a3 — кубический корень
Пример 1:

Найти: 83

83 = 2 потому что 23 = 8
Пример 2:

Найти: 814

814 = 3 потому что 34 = 81
Свойство 1

Корень n-й степени из произведения

Формулировка: Корень n-й степени из произведения неотрицательных множителей равен произведению корней n-й степени из этих множителей.

a⋅b n = an ⋅ bn

Условия: a≥0, b≥0, n∈ℕ, n>1

Пример 1:

Вычислить: 81⋅6254

81⋅6254 = 814 ⋅ 6254 = 3 ⋅ 5 = 15

Разложили на множители: 81=34 и 625=54

Пример 2:

Вычислить: 0.00032⋅31255

0.00032⋅31255 = 0.000325 ⋅ 31255 = 0.2 ⋅ 5 = 1

0.00032=0.25 и 3125=55

Свойство 2

Корень n-й степени из частного

Формулировка: Корень n-й степени из частного равен частному корней n-й степени делимого и делителя.

ab n = an bn

Условия: a≥0, b>0, n∈ℕ, n>1

Пример 1:

Вычислить: 0.06258100004

0.06258100004 = 0.06254 8100004 = 0.530 = 160
Пример 2 (сложный):

Вычислить: 2160000.0013

2160000.0013 = 2160003 0.0013 = 600.1 = 600

216000=603 и 0.001=0.13

Свойство 3

Корень из степени

Формулировка: Значение корня из степени не изменится, если и показатель корня, и показатель подкоренного выражения умножить на одно и то же натуральное число или разделить на их общий делитель.

am n = am⋅k n⋅k = am:k n:k
Пример 1:

Упростить: 1626

1626 = 286 = 243 = 163

Показатели 8 и 6 имеют общий делитель 2, поэтому: 86=43

Пример 2:

Упростить: 2524

2524 = 544 = 52 = 5

НОД(4, 4) = 4, поэтому: 44=11

Свойство 4

Корень из корня

Формулировка: Чтобы извлечь корень k-й степени из корня n-й степени из неотрицательного числа, достаточно извлечь корень степени n·k из этого числа.

an k = a n⋅k

Условия: a≥0, n>1, k>1

Пример 1:

Упростить: 6432

6432 = 643⋅2 = 646 = 266 = 2
Пример 2:

Упростить: 8124

8124 = 812⋅4 = 818 = 348 = 32 = 3
Свойство 5

Корень чётной и нечётной степени

Формулировка: Для любого действительного числа a и натурального n > 1:

an n = { |a| если  n — чётное число a если  n — нечётное число
Пример 1 (чётная степень):

Вычислить: (-5)66

(-5)66 = |-5| = 5

6 — чётное число, поэтому результат — модуль числа

Пример 2 (нечётная степень):

Вычислить: (-7)1313

(-7)1313 = -7

13 — нечётное число, поэтому результат равен самому числу

Пример 3:

Упростить: k1212, если k≥0

k1212 = |k| = k (так как k ≥ 0 )
Пример 4 (сложный):

Упростить: p1414, если p≤0

p1414 = |p| = -p (так как p ≤ 0 )

Практическое применение свойств корней

Задание 1

Вынесение множителя за знак корня

Правило: Если подкоренное выражение содержит степень с показателем, кратным показателю корня, то этот множитель можно вынести за знак корня.

ank n = ak (при a ≥ 0 )
Пример 1:

Вынесите множитель за знак корня: a5b25

a5b25 = a55 ⋅ b25 = a b25

Так как a5 — полная пятая степень, множитель a выносится полностью

Пример 2 (с условием):

Вынесите множитель за знак корня: a4b34 при a<0

a4b34 = a44 ⋅ b34 = |a| ⋅ b34 = -a ⋅ b34

Так как a<0 и степень корня чётная, выносится модуль |a|, который равен -a

Задание 2

Внесение множителя под знак корня

Правило: Чтобы внести множитель под знак корня n-ой степени, нужно возвести его в n-ую степень и умножить на подкоренное выражение.

k an = kn ⋅ a n
Пример 1:

Внесите множитель под знак корня: -223

-2 23 = - 23 ⋅ 2 3 = - 24 3 = - 16 3
Пример 2 (сложный):

Внесите множитель под знак корня: 2a-a4

2a -a4 = - ( -2a ) -a4 = - (-2a)4 ⋅ (-a) 4 = - 16 a4 ⋅ (-a) 4 = - -16 a5 4

Обратите внимание на знаки при внесении отрицательного множителя под корень чётной степени

Задание 3

Упрощение выражений с корнями

Принцип: Сложение и вычитание корней возможно только при одинаковых подкоренных выражениях и одинаковых показателях корней.

Пример 1:

Упростите выражение: 163+543-23

163 + 543 - 23 = 8⋅23 + 27⋅23 - 23 = 2 23 + 3 23 - 23 = 4 23

Сначала выносим множители, затем приводим подобные слагаемые

Пример 2:

Упростите выражение: (254-274)⋅34-2154

( 254 - 274 ) ⋅ 34 - 2154 = 2 5⋅34 - 27⋅34 - 2154 = 2154 - 814 - 2154 = - 814 = - 3
Задание 4

Умножение выражений с корнями

Формула разности квадратов: (x-y)(x+y)=x2-y2

Пример:

Выполните действия: (a-b4)(a+b4)(a+b)

( a - b4 ) ( a + b4 ) ( a + b ) = [ (a)2 - (b4)2 ] ( a + b ) = ( a - b ) ( a + b ) = a2 - b

Пояснение шагов:
1. Применили формулу разности квадратов к первым двум скобкам:
(a)2=a, (b4)2=b2/4=b1/2=b
2. Получили: a-b
3. Снова применили формулу разности квадратов к (a-b)(a+b)
4. Итог: a2-b

Свойства степенных функций с корнями

Функция 1

Функция y=x2k, где k∈ℕ

Общий вид: Корень чётной степени из x

y = x 2k , k ∈ ℕ

Примеры: y=x (k=1), y=x4 (k=2), y=x6 (k=3)

Свойства функции

1. Область определения

D ( f ) = [ 0 ; + ∞ )

По свойству арифметического корня чётной степени

2. Множество значений

E ( y ) = [ 0 ; + ∞ )

Функция принимает все неотрицательные значения

3. Нули функции

x 0 = 0

Единственный ноль при x = 0

4. Промежутки знакопостоянства

  • y>0 при x∈(0;+∞)
  • y=0 при x=0

5. Монотонность

Возрастает на всей области определения

Если 0≤x1<x2, то x12k<x22k

6. Чётность

Не является ни чётной, ни нечётной

Область определения не симметрична относительно нуля

График функции

Графики функций y=x2=x, y=x4, y=x6:

Все графики проходят через точки (0,0) и (1,1). Чем больше показатель корня, тем медленнее рост функции.

Функция 2

Функция y=x2k+1, где k∈ℕ

Общий вид: Корень нечётной степени из x

y = x 2k+1 , k ∈ ℕ

Примеры: y=x3 (k=1), y=x5 (k=2), y=x7 (k=3)

Свойства функции

1. Область определения

D ( f ) = ( - ∞ ; + ∞ )

Корень нечётной степени определён для всех действительных чисел

2. Множество значений

E ( y ) = ( - ∞ ; + ∞ )

Функция принимает все действительные значения

3. Нули функции

x 0 = 0

Единственный ноль при x = 0

4. Промежутки знакопостоянства

  • y>0 при x∈(0;+∞)
  • y<0 при x∈(-∞;0)
  • y=0 при x=0

5. Монотонность

Возрастает на всей области определения

Если x1<x2, то x12k+1<x22k+1

6. Чётность

Функция нечётная

y(-x)=-x2k+1=-x2k+1=-y(x)

График функции

Графики функций y=x3, y=x5:

Графики симметричны относительно начала координат. Проходят через точки (-1,-1), (0,0), (1,1). Чем больше показатель корня, тем ближе график к оси OX при |x| < 1 и тем дальше от оси OX при |x| > 1.

Уравнение вида f(x)±g(x)=a

Метод возведения в квадрат:

f(x) ± g(x) = a

1. Уединить один из корней
2. Возвести обе части в квадрат
3. Повторить при необходимости

x+3 = 1 + x-1 ⇒ x + 3 = 1 + 2 x-1 + x - 1

Проверка показывает, что корень подходит.
