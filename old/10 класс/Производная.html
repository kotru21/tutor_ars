Производная функции

Определение: Производной функции f(x) в точке x₀ называется предел отношения приращения функции к приращению аргумента при стремлении приращения аргумента к нулю:

f ′ ( x 0 ) = lim Δx → 0 f ( x 0 + Δx ) − f ( x 0 ) Δx

Обозначения: f′(x), y′, dfdx, dydx

Физический смысл: мгновенная скорость изменения функции

Графическая интерпретация:

Производная в точке x₀ равна угловому коэффициенту касательной к графику функции в этой точке

Пример

Вычисление производной функции f(x) = x² по определению

Решение:
1. f ( x 0 ) = x 0 2 2. f ( x 0 + Δx ) = ( x 0 + Δx ) 2 = x 0 2 + 2 x 0 Δx + ( Δx ) 2 3. f ( x 0 + Δx ) − f ( x 0 ) = 2 x 0 Δx + ( Δx ) 2 4. f ( x 0 + Δx ) − f ( x 0 ) Δx = 2 x 0 Δx + ( Δx ) 2 Δx = 2 x 0 + Δx 5. f ′ ( x 0 ) = lim Δx → 0 ( 2 x 0 + Δx ) = 2 x 0 6. Ответ: f ′ ( x ) = 2 x

Основные правила вычисления производных

1. Производная константы

( c )′ = 0 , где c = const
Пример:
( 5 )′ = 0

2. Производная степенной функции

( x n )′ = n x n − 1
Примеры:
( x 3 )′ = 3 x 2 ( x )′ = ( x 1 / 2 )′ = 1 2 x − 1 / 2 = 1 2 x

3. Производная суммы (разности)

( u ( x ) ± v ( x ) )′ = u ′ ( x ) ± v ′ ( x )
Пример:
( x 3 + 2 x 2 − 5 x + 3 )′ = 3 x 2 + 4 x − 5

4. Производная произведения

( u ( x ) ⋅ v ( x ) )′ = u ′ ( x ) ⋅ v ( x ) + u ( x ) ⋅ v ′ ( x )
Пример:
( x ⋅ ( x 2 + 1 ) )′ = x ′ ⋅ ( x 2 + 1 ) + x ⋅ ( x 2 + 1 )′ = 1 ⋅ ( x 2 + 1 ) + x ⋅ 2 x = x 2 + 1 + 2 x 2 = 3 x 2 + 1

5. Производная частного

( u ( x ) v ( x ) )′ = u ′ ( x ) ⋅ v ( x ) − u ( x ) ⋅ v ′ ( x ) [ v ( x ) ] 2
Пример:
( x x + 1 )′ = x ′ ⋅ ( x + 1 ) − x ⋅ ( x + 1 )′ ( x + 1 ) 2 = 1 ⋅ ( x + 1 ) − x ⋅1 ( x + 1 ) 2 = x + 1 − x ( x + 1 ) 2 = 1 ( x + 1 ) 2

Монотонность. Экстремумы

Применение

Исследование функции на монотонность

Теорема:

  1. Если f′(x)>0 на интервале (a, b), то f(x) возрастает на этом интервале
  2. Если f′(x)<0 на интервале (a, b), то f(x) убывает на этом интервале
Графики возрастающей и убывающей функций:
Экстремумы

Точки экстремума функции

Определения:

  • Точка экстремума: точки минимума и максимума
  • Точка максимума: точка x₀, в которой f'(x₀) = 0 и производная меняет знак с "+" на "-"
  • Точка минимума: точка x₀, в которой f'(x₀) = 0 и производная меняет знак с "-" на "+"
  • Экстремум: максимум или минимум функции
  • Максимум функции: значение f(x) в точке максимума
  • Минимум функции: значение f(x) в точке минимума
Пример 1: Определение точек экстремума по графику
Точки минимума:  x 1 , x 3 , x 5 , x 7 Значения: − 8 , − 4 , − 1 , 1 Точки максимума:  x 2 , x 4 , x 6 Значения: − 5 , − 3 , 0 Минимумы функции: − 4 , 1 , − 3 , − 4 Максимумы функции:  2 , 2.2 , − 2
Важное замечание:

Если функция не имеет производной в точке, но непрерывна в ней, эта точка может быть точкой экстремума.

f ( x ) = | x | , x 0 = 0 − точка минимума

Функция |x| имеет минимум в точке x=0, но производной в этой точке не существует (излома)

Пример 1

Нахождение промежутков монотонности функции

Дано: f(x)=1+3x2+x33−x44
Решение:
1. D ( f ) = ℝ  (все действительные числа)
2. Находим производную:
f ′ ( x ) = 0 + 6 x + x2 − x3 = − x3 + x2 + 6 x
f ′ ( x ) = − x ( x2 − x − 6 ) = − x ( x − 3 ) ( x + 2 )
3. Находим критические точки:
f ′ ( x ) = 0  при  − x ( x − 3 ) ( x + 2 ) = 0
x = − 2 , x = 0 , x = 3
Схема знаков производной:
4. Определяем экстремумы:
Точка минимума: x min = − 0 ( f ′ меняет знак с − на + ) Точка максимума: x max = 3 ( f ′ меняет знак с + на − ) Точка максимума: x max = -2 ( f ′ меняет знак с + на − )
5. Вычисляем значения функции (максимумы функции) в точках максимума:
f max ( − 2 ) = 1 + 3 ⋅ 4 + (−8) 3 − 16 4 = 1 + 12 − 83 − 4 = 9 − 83 = 273 − 83 = 193 f max ( 3 ) = 1 + 3 ⋅ 9 + 27 3 − 81 4 = 1 + 27 + 9 − 814 = 37 − 814 = 1484 − 814 = 674
Ответ:
Возрастает на: ( − ∞ ; − 2 ]  и  [ 0 ; 3 ] Убывает на: [ − 2 ; 0 ]  и  [ 3 ; + ∞ ) Точка максимума: x max = − 2 , f max = 193 Точка максимума: x max = 3 , f max = 674
Пример 2

Нахождение точек экстремума функции

Дано: f(x)=3x+x2x−1
Решение:
1. D ( f ) = ( − ∞ ; 1 ) ∪ ( 1 ; + ∞ )
2. f ′ ( x ) = ( 3 + 2 x ) ( x − 1 ) − ( 3 x + x 2 ) ⋅ 1 ( x − 1 ) 2 = x 2 − 2 x − 3 ( x − 1 ) 2 = ( x − 3 ) ( x + 1 ) ( x − 1 ) 2
3. f ′ ( x ) = 0  при  x = − 1  и  x = 3
Схема знаков производной:
4. Определяем экстремумы:
Точка максимума: x max = − 1 ( f ′  меняет знак с  +  на  − ) Точка минимума: x min = 3 ( f ′  меняет знак с  −  на  + )
Вычисляем экстремумы:
f max = f ( − 1 ) = 3 ( − 1 ) + ( − 1 ) 2 − 1 − 1 = − 3 + 1 − 2 = − 2 − 2 = 1 f min = f ( 3 ) = 3 ⋅ 3 + 3 2 3 − 1 = 9 + 9 2 = 18 2 = 9
Ответ:
Точка максимума: x max = − 1 f max = 1 Точка минимума: x min = 3 f min = 9
Пример 4

Наибольшее и наименьшее значение функции на отрезке

Дано: f(x)=−2x3−6x2+5 на отрезке [−1;1]
Шаг 1: Находим производную
f′(x)=−6x2−12x
Шаг 2: Критические точки
−6x2−12x=0 x2+2x=0 x(x+2)=0 x1=0, x2=−2
Шаг 3: Выбираем точки из отрезка
x1=0∈[−1;1], x2=−2∉[−1;1]
Шаг 4: Вычисляем значения функции
f(0)=−2⋅03−6⋅02+5=5 f(−1)=−2⋅(−1)3−6⋅(−1)2+5=1 f(1)=−2⋅13−6⋅12+5=−3
Графическое представление:
Отрезок [-1; 1]
Критическая точка x = 0
Максимум: f(0) = 5
Минимум: f(1) = -3
Шаг 5: Находим наибольшее и наименьшее значения
max[−1;1]f(x)=f(0)=5 min[−1;1]f(x)=f(1)=−3
Вывод:

На отрезке [-1; 1] функция достигает:

  • Наибольшего значения 5 в точке x = 0 (внутренняя критическая точка)
  • Наименьшего значения -3 в точке x = 1 (правый конец отрезка)

Заметим, что наименьшее значение находится не в критической точке, а на конце отрезка — это важно при решении задач на нахождение экстремумов на отрезке.
