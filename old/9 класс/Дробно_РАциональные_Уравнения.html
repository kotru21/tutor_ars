Дробно-рациональные уравнения

Основное свойство: равенство дроби нулю

Дробь равна нулю тогда и только тогда, когда её числитель равен нулю, а знаменатель не равен нулю.

P(x) Q(x) = 0 ⇔ { P(x)=0 Q(x)≠0

Это ключевое свойство, которое позволяет решать дробно-рациональные уравнения.

Пример 1: Простое дробно-рациональное уравнение

Решите уравнение: x-6x+6=0

x-6 x+6 = 0 ⇔ { x-6=0 x+6≠0 ⇔ { x=6 x≠-6 ⇔ x=6

Ответ: x=6

Пример 2: Уравнение с разложением на множители

Решите уравнение: x2-36x-6=0

x2-36 x-6 = 0 ⇔ { x2-36=0 x-6≠0

Разложим числитель на множители:

x2-36=0 ⇔ (x-6)(x+6)=0 ⇔ x=6илиx=-6

Учитываем ограничение:

x-6≠0⇒x≠6

Из двух корней оставляем только тот, который удовлетворяет ограничению:

x=-6

Ответ: x=-6

Пример 3: Уравнение с двумя корнями

Решите уравнение: x2-6xx+6=0

x2-6x x+6 = 0 ⇔ { x2-6x=0 x+6≠0

Решаем уравнение числителя:

x2-6x=0 ⇔ x(x-6)=0 ⇔ x=0илиx=6

Проверяем ограничение:

x+6≠0⇒x≠-6

Оба корня удовлетворяют ограничению (ни 0, ни 6 не равны -6).

x=0,x=6

Ответ: x=0;x=6

Пример 4: Уравнение без решений

Решите уравнение: x+6x2-36=0

x+6 x2-36 = 0 ⇔ { x+6=0 x2-36≠0

Решаем уравнение числителя:

x+6=0 ⇔ x=-6

Решаем ограничение:

x2-36≠0 ⇔ (x-6)(x+6)≠0 ⇔ x≠6иx≠-6

Корень числителя x=-6 НЕ удовлетворяет ограничению, так как он обращает знаменатель в ноль.

x=-6∉ОДЗ

Ответ: Нет корней

Пример 5: Уравнение с несколькими дробями (подробное решение)

Решите уравнение: xx2-5x+6+3x-2=2x-3

Шаг 1: Найдём ОДЗ (область допустимых значений)

Для каждой дроби знаменатель не должен быть равен нулю:

{ x2-5x+6≠0 (для первой дроби) x-2≠0 (для второй дроби) x-3≠0 (для третьей дроби)

Шаг 2: Решим каждое неравенство

1) x2-5x+6≠0

Сначала найдем корни квадратного уравнения:

x2-5x+6=0
D=52-4·1·6=25-24=1
x1=5-12=2, x2=5+12=3

Значит:

x2-5x+6=(x-2)(x-3)

Условие x2-5x+6≠0 означает:

x≠2иx≠3

2) x-2≠0⇒x≠2

3) x-3≠0⇒x≠3

Шаг 3: Объединим все ограничения

Все три условия дают одно и то же ограничение: x≠2 и x≠3

ОДЗ:x∈ℝ\{2,3}

Шаг 4: Приведем все дроби к общему знаменателю

Заметим, что:

x2-5x+6=(x-2)(x-3)

Поэтому общий знаменатель для всех трех дробей: (x-2)(x-3)

Запишем уравнение с общим знаменателем:

x(x-2)(x-3) + 3(x-3)(x-2)(x-3) = 2(x-2)(x-2)(x-3)

Шаг 5: Умножим обе части уравнения на общий знаменатель

Так как x≠2 и x≠3 (из ОДЗ), знаменатель не равен нулю, поэтому умножение допустимо:

x+3(x-3)=2(x-2)

Шаг 6: Решим полученное линейное уравнение

x+3x-9=2x-4
4x-9=2x-4

Перенесем все члены с x влево, а числа вправо:

4x-2x=9-4
2x=5
x=2.5

Или в виде дроби: x=52

Шаг 7: Проверим, удовлетворяет ли найденный корень ОДЗ

Нам нужно проверить:

  • x=2.5≠2 ✓ (верно)
  • x=2.5≠3 ✓ (верно)

Корень удовлетворяет всем ограничениям ОДЗ.

Шаг 8: Сделаем проверку подстановкой (по желанию)

Подставим x = 2.5 в исходное уравнение:

2.52.52-5·2.5+6 + 32.5-2 = 22.5-3
2.56.25-12.5+6 + 30.5 = 2-0.5
2.5-0.25 + 6 = -4
-10+6=-4
-4=-4

Равенство верно, проверка подтверждает правильность решения.

Ответ: x=2.5 или x=52

Пояснение: В этом уравнении важно было:

  1. Найти ОДЗ и исключить значения, обращающие знаменатели в ноль
  2. Найти общий знаменатель для всех дробей
  3. Умножить на общий знаменатель (только после проверки, что он не равен нулю!)
  4. Решить полученное уравнение
  5. Проверить, что найденный корень принадлежит ОДЗ

Пример 6: Нахождение нулей функции с разложением на множители

Найдите нули функции: f(x)=x3-7x2+12xx-4

Шаг 1: Найдем ОДЗ

Знаменатель не должен быть равен нулю:

x-4≠0⇒x≠4

Шаг 2: Найдем нули функции

Нули функции — это значения x, при которых f(x) = 0. Дробь равна нулю, когда числитель равен нулю, а знаменатель не равен нулю.

f(x)=0⇔ { x3-7x2+12xx-4=0 x≠4

Упростим числитель, вынеся общий множитель:

x3-7x2+12x= x(x2-7x+12)

Разложим квадратный трехчлен на множители:

x2-7x+12=0
D=72-4·1·12=49-48=1
x1=7-12=3, x2=7+12=4

Получаем разложение:

x2-7x+12=(x-3)(x-4)

Теперь исходная функция:

f(x)= x(x-3)(x-4)x-4

Важно: Нельзя просто сократить (x-4), так как при x=4 функция не определена! Но для нахождения нулей можно рассмотреть числитель:

x(x-3)(x-4)=0

Получаем корни: x=0, x=3, x=4

Шаг 3: Проверим корни на принадлежность ОДЗ

Из ОДЗ: x ≠ 4

Поэтому корень x=4 не входит в область определения и не является нулем функции.

Шаг 4: Запишем ответ

x=0,x=3

Ответ: Нули функции: x=0 и x=3

Пример 7: Уравнение, решаемое методом замены переменной

Решите уравнение: 2x-1x+x2x-1=4.25

Шаг 1: Найдем ОДЗ

{ x≠0 2x-1≠0⇒x≠0.5

Шаг 2: Сделаем замену переменной

Заметим, что дроби взаимно обратны. Введем замену:

t=2x-1x

Тогда:

x2x-1=1t

Преобразуем 4.25 в дробь:

4.25=4+14=174

Исходное уравнение принимает вид:

t+1t=174

Шаг 3: Решим полученное уравнение

Умножим обе части на 4t (t ≠ 0 из ОДЗ):

4t2+4=17t
4t2-17t+4=0

Найдем дискриминант:

D=172-4·4·4=289-64=225
D=15
t1=17-158=28=14, t2=17+158=328=4

Шаг 4: Вернемся к переменной x

Случай 1: t = 1/4

2x-1x=14

Умножим крест-накрест:

4(2x-1)=x
8x-4=x
8x-x=4
7x=4
x=47

Случай 2: t = 4

2x-1x=4
2x-1=4x
2x-4x=1
-2x=1
x=-12

Шаг 5: Проверим корни на принадлежность ОДЗ

ОДЗ: x ≠ 0 и x ≠ 0.5

Оба найденных корня удовлетворяют ОДЗ:

  • x = 4/7 ≈ 0.571 ≠ 0 и ≠ 0.5
  • x = -1/2 = -0.5 ≠ 0 и ≠ 0.5

Ответ: x=47,x=-12
