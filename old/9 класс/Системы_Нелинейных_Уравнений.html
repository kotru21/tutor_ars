Системы нелинейных уравнений

Что такое система уравнений и ее решение

Система уравнений — это набор двух или более уравнений, которые рассматриваются одновременно.

Решение системы уравнений — это упорядоченная пара чисел (x; y), которая удовлетворяет всем уравнениям системы одновременно.

Геометрически решение системы представляет собой точку пересечения графиков уравнений, входящих в систему.

Метод подстановки

Метод подстановки

Метод подстановки — метод решения систем уравнений, при котором из одного уравнения выражают одну переменную через другую и подставляют это выражение в другое уравнение.

Пример: Решение системы методом подстановки

Решите систему уравнений:

{ x+y=3 x2+y2=9

Решение:

1. Выразим x из первого уравнения:

x=3-y

2. Подставим это выражение во второе уравнение:

(3-y)2+y2=9

3. Раскроем скобки и упростим:

9-6y+y2+y2=9
2y2-6y=0
2y(y-3)=0

4. Решим полученное уравнение:

y=0илиy=3

5. Найдем соответствующие значения x:

При y=0:x=3-0=3
При y=3:x=3-3=0

Ответ: (3; 0) и (0; 3)

Метод сложения

Метод сложения

Метод сложения (исключения) — метод решения систем уравнений, при котором уравнения системы складывают или вычитают таким образом, чтобы исключить одну из переменных.

Пример: Решение системы методом сложения

Решите систему уравнений:

{ y2+xy=-10 5y-2xy=17

Решение:

1. Перепишем систему:

{ xy+y2=-10 -2xy+5y=17

2. Умножим первое уравнение на 2:

2xy+2y2=-20

3. Сложим полученное уравнение со вторым уравнением системы:

(2xy+2y2)+(-2xy+5y)=-20+17
2y2+5y=-3

4. Решим полученное квадратное уравнение:

2y2+5y+3=0
D=52-4·2·3=25-24=1
y1=-5+12·2=-44=-1
y2=-5-12·2=-64=-32=-1,5

5. Найдем соответствующие значения x из первого уравнения системы:

При y = -1:

x·(-1)+(-1)2=-10
-x+1=-10
x=11

При y = -3/2:

x·(-32)+(-32)2=-10
-3x2+94=-10
-3x2=-10-94=-494
x=496

Ответ: (11; -1) и (49/6; -3/2)

Графический метод

Графический метод

Графический метод — метод решения систем уравнений, при котором строят графики уравнений, входящих в систему, и находят точки их пересечения.

Пример: Решение системы графическим методом

Решите графически систему уравнений:

{ y-x-2=0 y-(x+1)2=-1

Решение:

1. Преобразуем уравнения к виду, удобному для построения графиков:

y=x+2
y=(x+1)2-1

2. Построим графики:

  • Прямая y = x + 2 — линейная функция, проходящая через точки (0; 2) и (-2; 0)
  • Парабола y = (x + 1)² - 1 — парабола, ветви вверх, вершина в точке (-1; -1)

3. На графике видно две точки пересечения:

A(-2;0)иB(1;3)

4. Проверим решение аналитически:

Для точки A(-2; 0):

0-(-2)-2=0+2-2=0✓
0-(-2+1)2=0-(-1)2=0-1=-1✓

Для точки B(1; 3):

3-1-2=3-3=0✓
3-(1+1)2=3-22=3-4=-1✓

Ответ: (-2; 0) и (1; 3)
