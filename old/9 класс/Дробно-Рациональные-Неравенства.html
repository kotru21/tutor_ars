Метод интервалов: общий алгоритм

Метод интервалов — это универсальный способ решения неравенств вида:

  • P(x)>0 (или ≥0, <0, ≤0)
  • Где P(x) — произведение или частное линейных множителей

Алгоритм решения:

  1. Найти нули функции — точки, где каждый множитель равен нулю
  2. Отметить эти точки на числовой прямой, разбивая её на интервалы
  3. Определить знак функции на каждом интервале (методом подстановки или анализа множителей)
  4. Выбрать нужные интервалы в соответствии со знаком неравенства
  5. Записать ответ с учётом строгости неравенства

Пример 1: Решение неравенства (x−1)(x+2,5)(x−4)<0

Решение:

1. Находим нули функции: каждый множитель приравниваем к нулю

x−1=0⇒x=1
x+2.5=0⇒x=−2.5
x−4=0⇒x=4

2. Отмечаем точки на числовой прямой в порядке возрастания:

3. Определяем знаки на интервалах:

  • (-∞, -2.5): берём x=−3 → (−)(−)(−)=(−)<0
  • (-2.5, 1): берём x=0 → (−)(+)(−)=(+)>0
  • (1, 4): берём x=2 → (+)(+)(−)=(−)<0
  • (4, +∞): берём x=5 → (+)(+)(+)=(+)>0

4. Выбираем интервалы, где выражение < 0 (отрицательно):

Ответ:

x∈(−∞,−2.5)∪(1,4)

Решение неравенства (x+9)²(x−2)(x−3) ≤ 0

Решение:

Дано неравенство: (x+9)²(x−2)(x−3) ≤ 0

Особенности:

  1. Множитель (x+9)² имеет чётную степень (квадрат)
  2. Неравенство нестрогое (≤), поэтому точки, где выражение равно нулю, включаются в ответ

1. Находим нули функции:

(x+9)2=0⇒x=−9
x−2=0⇒x=2
x−3=0⇒x=3

2. Располагаем нули на числовой прямой в порядке возрастания:

3. Правило смены знака:

  • В точке x = -9 (чётная кратность) — знак НЕ меняется
  • В точках x = 2 и x = 3 (нечётная кратность) — знак меняется

4. Определяем знаки на интервалах:

  • Интервал (-∞, -9): Выбираем x = -10:
    (-10+9)² = (-1)² = (+) > 0
    (-10-2) = (-12) = (-) < 0
    (-10-3) = (-13) = (-) < 0
    (+)(-)(-) = (+) > 0
  • Интервал (-9, 2): Выбираем x = 0:
    (0+9)² = (9)² = (+) > 0
    (0-2) = (-2) = (-) < 0
    (0-3) = (-3) = (-) < 0
    (+)(-)(-) = (+) > 0
    Знак не изменился, так как точка -9 имеет чётную кратность!
  • Интервал (2, 3): Выбираем x = 2.5:
    (2.5+9)² = (11.5)² = (+) > 0
    (2.5-2) = (0.5) = (+) > 0
    (2.5-3) = (-0.5) = (-) < 0
    (+)(+)(-) = (-) < 0
  • Интервал (3, +∞): Выбираем x = 4:
    (4+9)² = (13)² = (+) > 0
    (4-2) = (2) = (+) > 0
    (4-3) = (1) = (+) > 0
    (+)(+)(+) = (+) > 0

5. Выбираем интервалы, где выражение ≤ 0:

Нас интересуют интервалы, где выражение отрицательно или равно нулю:

  • На интервале (2, 3): выражение отрицательно
  • В точках x = -9, x = 2, x = 3: выражение равно нулю
  • На остальных интервалах выражение положительно

Ответ:

x∈[2,3]∪{-9}

Объяснение:
1. Интервал [2, 3] — квадратные скобки означают, что точки 2 и 3 включены (неравенство нестрогое ≤)
2. Точка -9 также включается, так как в этой точке выражение равно нулю
3. Хотя на интервале (-9, 2) выражение тоже равно нулю в точке -9, сам интервал не входит в решение, так как на всём интервале (-9, 2) выражение положительно

Пример 5: Решение неравенства (x−4)2(3−x)x+1≥0

Правильное решение:

Дано: (x−4)2(3−x)x+1≥0

3−x=−(x−3)

(x−4)2·[−(x−3)]x+1≥0

При умножении неравенства на отрицательное число направление неравенства меняется на противоположное:

(x−4)2(x−3)x+1≤0

Теперь мы получили эквивалентное неравенство в стандартном виде!

Шаг 8: Выбираем решение для неравенства f(x) ≤ 0:

Нас интересуют интервалы, где:

  1. f(x) < 0 (отрицательные значения)
  2. f(x) = 0 (нули числителя)

Из диаграммы видно:

  • f(x) < 0 только на интервале (-1, 3)
  • f(x) = 0 в точках x = 3 и x = 4
  • Точка x = -1 не входит в ОДЗ

Так как неравенство нестрогое (≤), включаем точки, где f(x) = 0.

Итоговый ответ:

x∈(−1,3]∪{4}

Пояснение:
1. (-1, 3] — интервал от -1 до 3:
• Круглая скобка у -1: точка -1 не входит (знаменатель = 0)
• Квадратная скобка у 3: точка 3 входит (числитель = 0, неравенство ≥ 0)
• На всём интервале (-1, 3) выражение отрицательно

2. {4} — отдельная точка:
• В точке x = 4: числитель = 0 (так как (x-4)² = 0)
• Неравенство нестрогое (≥), поэтому точка включается
• Но точка 4 не образует интервал, так как знак не меняется при переходе через неё (чётная кратность)

Проверка решения:

Для x = 0 ∈ (-1, 3]:

(0−4)2(3−0)0+1= 16·31=48≥0✓

Для x = 3 ∈ (-1, 3]:

(3−4)2(3−3)3+1= 1·04=0≥0✓

Для x = 4 ∈ {4}:

(4−4)2(3−4)4+1= 0·(−1)5=0≥0✓

Для x = -2 ∉ решению:

(−2−4)2(3−(−2))−2+1= 36·5−1=−180≱0✓

Пример 3: Решение дробно-рационального неравенства

Решите неравенство:

(x+3)(x-1) (x-4)(x+2) ≥0

Шаг 1: Находим критические точки

Разложим функцию на множители и найдем точки, где она меняет знак:

Нули числителя (дробь равна 0):
(x+3)(x-1)=0
x=-3 или x=1
Нули знаменателя (дробь не определена):
(x-4)(x+2)=0
x=4 или x=-2

Важно: Нули числителя включаются в решение при нестрогом неравенстве (≥), а нули знаменателя никогда не включаются (там функция не определена).

Шаг 2: Отмечаем точки на числовой прямой

Располагаем точки в порядке возрастания:

-3    -2    1    4
  • ● — закрашенная точка (включается в решение)
  • ○ — выколотая точка (не включается в решение)

Шаг 3: Определяем знаки на интервалах

Получаем 5 интервалов:

(-∞,-3), (-3,-2), (-2,1), (1,4), (4,+∞)

Проверим знак дроби в каждом интервале:

Интервал Пробная точка Знак дроби
(-∞, -3) x = -4 (-)(-) (-)(-) =+
(-3, -2) x = -2.5 (-)(-) (-)(+) =-
(-2, 1) x = 0 (+)(-) (-)(+) =+
(1, 4) x = 2 (+)(+) (-)(+) =-
(4, +∞) x = 5 (+)(+) (+)(+) =+

Шаг 4: Записываем решение

Нас интересуют интервалы, где дробь ≥ 0 (положительна или равна нулю).

Положительные интервалы: (-∞, -3), (-2, 1), (4, +∞)

Добавляем точки, где дробь равна нулю (нули числителя):

  • Точка -3: включаем (закрытая скобка ]
  • Точка 1: включаем (закрытая скобка ]

Не включаем точки -2 и 4 (знаменатель = 0).

Ответ:

x∈(-∞,-3]∪(-2,1]∪(4,+∞)
Пояснение:
  1. (-∞, -3] — включаем -3, так как при x = -3 дробь равна 0
  2. (-2, 1] — не включаем -2 (знаменатель = 0), включаем 1 (числитель = 0)
  3. (4, +∞) — не включаем 4 (знаменатель = 0)

Таким образом, неравенство выполняется для всех x из указанных промежутков.

Решение:

x(x−2)(x−3)+3x−2−2x−3>0

x+3x−9−2x+4(x−2)(x−3)>0

Числитель:2x−5=0⇒x=2.5
Знаменатель:(x−2)(x−3)=0⇒x=2 или x=3

Ответ:

x∈(2,2.5)∪(3,+∞)

Точки 2 и 3 не включаются (знаменатель = 0), точка 2.5 не включается (строгое неравенство).

Итоги метода интервалов

Ключевые моменты:

  1. Чётные степени множителей не меняют знак при переходе через соответствующую точку
  2. Точки из знаменателя всегда выколотые (не включаются в ответ)
  3. Точки из числителя включаются при нестрогих неравенствах (≥, ≤)
  4. Система неравенств — находим пересечение решений
  5. Совокупность неравенств — находим объединение решений
text

Проверка решения:

  • Подставить пограничные значения (точки, где выражение = 0)
  • Проверить знак на каждом интервале
  • Убедиться, что учтены все ограничения (знаменатель ≠ 0)
