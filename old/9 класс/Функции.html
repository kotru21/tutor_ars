Функция

Определение функции

Функцией называется правило, которое каждому элементу x из множества X ставит в соответствие единственный элемент y из множества Y.

f:X→Y

Обозначение: y=f(x), где:

  • x — аргумент (независимая переменная)
  • y — значение функции (зависимая переменная)
  • f — правило соответствия

График функции

График функции

Графиком функции y=f(x) называется множество всех точек плоскости с координатами (x,f(x)).

Основное свойство: Любая вертикальная прямая пересекает график функции не более чем в одной точке.

Работа со значениями функций

Определение значения функции в точке

Чтобы найти значение функции f(a):

  1. Находим точку a на оси OX
  2. Проводим вертикальную прямую до пересечения с графиком
  3. От точки пересечения проводим горизонтальную прямую к оси OY
  4. Координата на оси OY — значение функции
f(-4)=2
f(7)=-3

Нахождение аргумента по значению функции

Чтобы найти x, при котором f(x)=c:

  1. Находим точку c на оси OY
  2. Проводим горизонтальную прямую до пересечения с графиком
  3. От точек пересечения проводим вертикальные прямые к оси OX
  4. Координаты на оси OX — искомые значения аргумента
f(x)=3  при  x=-7,-5,-2,1

Область определения и множество значений

Область определения и множество значений

Область определения D(f) — множество всех значений аргумента x, при которых функция определена.

Множество значений E(f) — множество всех значений, которые принимает функция.

D(f)=[1,6]
E(f)=[1,5]

Нули функции

Нули функции

Нули функции — это значения аргумента x, при которых значение функции равно нулю.

Математически: f(x0)=0, где x0 — нуль функции.

Геометрический смысл: Нули функции — это абсциссы точек пересечения графика функции с осью OX.

f(x)=0 при  x=-2,4,8

На графике видно, что функция пересекает ось OX в трех точках: x=-2, x=4 и x=8.

Промежутки знакопостоянства

Промежутки знакопостоянства функции

Промежутками знакопостоянства называются интервалы значений аргумента, на которых функция сохраняет постоянный знак (положительный или отрицательный).

Функция положительна (f(x)>0):

Там, где график находится выше оси OX

x∈(-9,-2)∪(4,8)

Функция отрицательна (f(x)<0):

Там, где график находится ниже оси OX

x∈(-2,4)∪(8,9)

На концах промежутков функция равна нулю (в точках -2, 4, 8), поэтому эти точки не включаются в промежутки знакопостоянства.

Монотонность функции

Монотонность функции

Возрастающая функция: Функция f(x) называется возрастающей на промежутке, если для любых x1<x2 из этого промежутка выполняется неравенство f(x1)<f(x2).

Убывающая функция: Функция f(x) называется убывающей на промежутке, если для любых x1<x2 из этого промежутка выполняется неравенство f(x1)>f(x2).

На графике возрастающей функции при движении слева направо график "поднимается вверх". На графике убывающей функции при движении слева направо график "опускается вниз".

Пример анализа монотонности

Функция возрастает:

x∈(-9,-8)∪(-7.5,1.5)∪(6,9)

На этих промежутках график "поднимается" при движении слева направо

Функция убывает:

x∈(-8,-6.5)∪(1.5,6)

На этих промежутках график "опускается" при движении слева направо

Важные точки:

  • Точки максимума: x=-8 и x=1.5 (переход от возрастания к убыванию)
  • Точки минимума: x=-6.5 и x=6 (переход от убывания к возрастанию)

Четная функция

Определение четной функции

Функция y=f(x) называется четной, если для любого x из области определения выполняется:

1)x∈D(f)⇒-x∈D(f)
2)f(-x)=f(x)

Геометрический смысл: График четной функции симметричен относительно оси OY.

Если точка (x,y) принадлежит графику четной функции, то и точка (-x,y) также принадлежит графику.

Примеры графиков четных функций

Нечетная функция

Определение нечетной функции

Функция y=f(x) называется нечетной, если для любого x из области определения выполняется:

1)x∈D(f)⇒-x∈D(f)
2)f(-x)=-f(x)

Геометрический смысл: График нечетной функции симметричен относительно начала координат.

Если точка (x,y) принадлежит графику нечетной функции, то и точка (-x,-y) также принадлежит графику.

Примеры графиков нечетных функций

Алгоритм определения четности функции

  1. Проверить область определения: Должна быть симметрична относительно нуля (если x∈D(f), то -x∈D(f))
  2. Вычислить f(-x): Упростить выражение
  3. Сравнить с f(x):
    • Если f(-x)=f(x) — функция четная
    • Если f(-x)=-f(x) — функция нечетная
    • Если оба условия не выполняются — функция ни четная, ни нечетная

Преобразования графиков

Преобразования графиков функций

Графики функций можно перемещать (сдвигать) вдоль осей координат. Эти преобразования сохраняют форму графика, меняя только его положение.

Рассмотрим основные виды преобразований:

  1. Сдвиг вверх/вниз (параллельный перенос вдоль оси OY)
  2. Сдвиг влево/вправо (параллельный перенос вдоль оси OX)

Сдвиг графика вдоль оси OY (вверх/вниз)

Исходная функция: y=x

y=x+4 — сдвиг на 4 единицы вверх

Исходная функция: y=x

y=x-4 — сдвиг на 4 единицы вниз

Общее правило: График функции y=f(x)+a получается из графика y=f(x) сдвигом на a единиц вдоль оси OY.

  • Если a>0 — сдвиг вверх
  • Если a<0 — сдвиг вниз

Сдвиг графика вдоль оси OX (влево/вправо)

Исходная функция: y=2x2

y=2(x-6)2 — сдвиг на 6 единиц вправо

Исходная функция: y=1x

y=1x+2 — сдвиг на 2 единицы влево

Общее правило: График функции y=f(x-a) получается из графика y=f(x) сдвигом на a единиц вдоль оси OX.

  • Если y=f(x-a) — сдвиг на a единиц вправо
  • Если y=f(x+a) — сдвиг на a единиц влево

Важно запомнить: Знак в скобке противоположен направлению сдвига!

Задание 4: Построение графиков с помощью преобразований

С помощью преобразований графика функции y=|x| построите графики:

а) y=|x+4|

Решение:

Исходная функция: y=|x|

Преобразование: y=|x+4|=f(x+4)

Так как f(x+4) — это сдвиг на 4 единицы влево

График:

б) y=|x|-4

Решение:

Исходная функция: y=|x|

Преобразование: y=|x|-4=f(x)-4

Так как f(x)-4 — это сдвиг на 4 единицы вниз

График:

в) y=|x-3|+2

Решение:

Исходная функция: y=|x|

Два преобразования:

  1. y=|x-3| — сдвиг на 3 единицы вправо
  2. y=|x-3|+2 — сдвиг на 2 единицы вверх

Итого: сдвиг на 3 единицы вправо и на 2 единицы вверх

График:

Алгоритм построения графиков с помощью преобразований

  1. Определить исходную функцию (базовый график)
  2. Проанализировать преобразования:
    • Если f(x±a) — сдвиг вдоль OX
    • Если f(x)±a — сдвиг вдоль OY
  3. Определить направление сдвигов:
    • f(x+a) — сдвиг влево на a
    • f(x-a) — сдвиг вправо на a
    • f(x)+a — сдвиг вверх на a
    • f(x)-a — сдвиг вниз на a
  4. Выполнить преобразования последовательно (если их несколько)
  5. Построить график

Важно: При нескольких преобразованиях порядок выполнения иногда важен. Сначала выполняют сдвиги вдоль OX, затем вдоль OY.

Примеры сложных преобразований

y=x-2+3

Преобразования для y=x:

  • y=x-2 — сдвиг на 2 единицы вправо
  • y=x-2+3 — сдвиг на 3 единицы вверх
y=|x+1|-2

Преобразования для y=|x|:

  • y=|x+1| — сдвиг на 1 единицу влево
  • y=|x+1|-2 — сдвиг на 2 единицы вниз
